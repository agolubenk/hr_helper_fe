# Документация страницы: Редактирование заявки (hiring-requests/[id]/edit/page.tsx)

## Общее описание

Страница редактирования заявки на подбор. Позволяет редактировать все параметры заявки через форму.

## Компоненты страницы

### 1. Заголовок страницы
- **Компонент**: `Flex` с кнопкой "Назад"
- **Назначение**: Навигация и заголовок страницы
- **Логика**: 
  - Кнопка "Назад" возвращает на детальную страницу заявки
  - Отображает название страницы "Редактирование заявки"
- **Связи**: Навигация на `/hiring-requests/[id]`

### 2. Форма редактирования
- **Компонент**: Форма с полями
- **Назначение**: Редактирование данных заявки
- **Поля**:
  - Название вакансии
  - Грейд
  - Проект
  - Отдел
  - Рекрутер
  - Приоритет
  - Технологии
  - Сроки
  - И другие параметры
- **Логика**: 
  - Заполнение полей текущими данными заявки
  - Валидация обязательных полей
  - Сохранение изменений через API

## URL параметры

### [id]
- **Тип**: Динамический параметр маршрута
- **Назначение**: ID заявки для редактирования
- **Пример**: `/hiring-requests/123/edit` редактирует заявку с ID 123
- **Обработка**: 
  - Читается через `useParams()`
  - Используется для загрузки данных заявки

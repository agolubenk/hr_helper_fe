# Документация страницы: Редактирование оценки (recr-chat/.../assessment/[assessmentId]/edit/page.tsx)

## Общее описание

Страница редактирования ранее сохранённой оценки кандидата. Доступна в течение заданного времени с момента проведения оценки (EDIT_WINDOW_HOURS). Блоки и шкала оценок те же, что и на странице создания оценки.

## Параметры маршрута

- `vacancyId` — идентификатор вакансии.
- `candidateId` — идентификатор кандидата.
- `assessmentId` — идентификатор оценки.

## Компоненты страницы

- Форма по блокам компетенций с Select по шкале (из матрицы специализации вакансии) и кнопкой «Скриншот».
- При превышении окна редактирования отображается сообщение о недоступности редактирования.

## Связи с другими страницами

- `lib/ratingScale`: шкала по специализации вакансии.
- `/recr-chat/.../assessment/new` — та же структура блоков и шкала при создании.
- Мок связи вакансия → специализация: `VACANCY_SPEC_ID` (далее — из API).

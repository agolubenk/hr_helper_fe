# Документация страницы: Редактирование статьи вики (wiki/[id]/edit/page.tsx)

## Общее описание

Страница редактирования статьи вики. Позволяет редактировать содержимое статьи, добавлять разделы, изменять теги и категорию.

## Компоненты страницы

### 1. WikiEditForm
- **Расположение**: `@/components/wiki/WikiEditForm`
- **Назначение**: Форма редактирования статьи вики
- **Пропсы**:
  - `page`: данные статьи для редактирования
  - `onSave`: обработчик сохранения
  - `onCancel`: обработчик отмены
- **Логика**: 
  - Форма с полями: заголовок, категория, теги, содержимое
  - Редактор содержимого (markdown или WYSIWYG)
  - Валидация обязательных полей
  - Сохранение через API
- **Связи**: Используется для редактирования статьи

### 2. Заголовок страницы
- **Компонент**: `Flex` с кнопками
- **Назначение**: Навигация и действия
- **Логика**: 
  - Кнопка "Отмена" возвращает на детальную страницу
  - Кнопка "Сохранить" сохраняет изменения
- **Связи**: Навигация на `/wiki/[id]`

## Состояние компонента

### State переменные:
- `page`: данные статьи для редактирования
- `loading`: флаг загрузки данных
- `saving`: флаг сохранения изменений

## URL параметры

### [id]
- **Тип**: Динамический параметр маршрута
- **Назначение**: ID статьи для редактирования
- **Пример**: `/wiki/1/edit` редактирует статью с ID 1
- **Обработка**: 
  - Читается через `useParams()`
  - Используется для загрузки данных статьи

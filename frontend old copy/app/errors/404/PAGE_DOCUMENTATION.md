# Документация страницы: Ошибка 404 (errors/404/page.tsx)

## Общее описание

Страница ошибки 404 (Not Found - страница не найдена). Отображается при обращении к несуществующему URL. Также доступна через файл `not-found.tsx` в корне `app/` для автоматического использования Next.js.

## Компоненты страницы

### 1. Анимированный фон
- **Компонент**: `Box` с классом `background`
- **Назначение**: Создание визуально привлекательного фона с плавающими иконками
- **Логика**: 
  - Создает 40 плавающих иконок с анимацией
  - Иконки: иконки из меню приложения (связь с основным приложением)
  - Каждая иконка имеет уникальные параметры анимации

### 2. FloatingIcon компонент
- **Компонент**: `FloatingIcon` (локальный компонент)
- **Назначение**: Отображение анимированной иконки, плавающей по экрану
- **Параметры**:
  - `Icon`: компонент иконки из Radix UI
  - `index`: индекс иконки для определения параметров анимации
- **Логика**: Аналогична другим страницам ошибок (см. общую документацию)

### 3. Код ошибки
- **Компонент**: `Text` с классом `errorCode`
- **Назначение**: Отображение крупного кода ошибки (404)
- **Логика**: 
  - Градиентная заливка (синий accent-9)
  - Анимация пульсации
  - Адаптивный размер

### 4. Заголовок
- **Компонент**: `Flex` с иконкой `MagnifyingGlassIcon` и текстом
- **Назначение**: Отображение названия ошибки с визуальным акцентом
- **Логика**: 
  - Иконка поиска с анимацией вращения и покачивания
  - Текст "Страница не найдена"
  - Визуально подчеркивает, что страница не найдена

### 5. Описание ошибки
- **Компонент**: `Text` с описанием
- **Назначение**: Информирование пользователя об ошибке
- **Логика**: 
  - Текстовое описание о том, что страница не существует или была перемещена
  - Центрированный текст

### 6. Кнопки навигации
- **Компонент**: `Button` с навигацией
- **Назначение**: Предоставление способов возврата в приложение
- **Логика**: 
  - "На главную": переход на `/workflow`
  - "Назад": возврат в истории браузера (`router.back()`)
- **Связи**: 
  - Использует `useRouter` из Next.js для навигации

## Функции и обработчики

### FloatingIcon.useEffect
- **Назначение**: Инициализация параметров анимации для иконки
- **Логика**: Аналогична другим страницам ошибок

### NotFound
- **Назначение**: Главный компонент страницы ошибки 404
- **Логика**: Отображает код ошибки, описание и кнопки навигации

## Стили

- **Файл**: `not-found.module.css`
- **Основные классы**: Аналогичны другим страницам ошибок
- **Анимации**:
  - `@keyframes float` - плавание иконок
  - `@keyframes pulse` - пульсация кода ошибки
  - `@keyframes searchRotate` - вращение иконки поиска
  - `@keyframes searchWrapper` - покачивание обертки иконки поиска

## Связи с другими страницами

- `/workflow` - главная рабочая страница (кнопка "На главную")

## Особенности

- Статическая страница ошибки
- Анимированный фон с 40 плавающими иконками (больше, чем на других страницах)
- Уникальная анимация иконки поиска (вращение и покачивание)
- Цветовая схема: синий (accent-9)
- Используется Next.js для автоматического отображения при 404 ошибках
- Иконки из меню приложения создают связь с основным приложением

## Использование

Страница автоматически отображается при:
- Переходе на несуществующий URL
- Удалении страницы
- Неправильном вводе адреса
- Ошибках маршрутизации## ПримечаниеФайл `not-found.tsx` также находится в корне `app/` для автоматического использования Next.js при 404 ошибках. Файл в `errors/404/page.tsx` является копией для единообразия структуры.

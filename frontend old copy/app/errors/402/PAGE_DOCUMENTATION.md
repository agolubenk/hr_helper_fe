# Документация страницы: Ошибка 402 (error-402/page.tsx)

## Общее описание

Страница ошибки 402 (Payment Required). Отображается при необходимости оплаты подписки или при истечении подписки. Использует единый дизайн с другими страницами ошибок: анимированный фон с плавающими иконками, крупный код ошибки, и кнопки навигации.

**Путь**: `/errors/402`

## Компоненты страницы

### 1. Анимированный фон
- **Компонент**: `Box` с классом `background`
- **Назначение**: Создание визуально привлекательного фона с плавающими иконками
- **Логика**: 
  - Создает 35 плавающих иконок с анимацией
  - Иконки: звезды, галочки, запреты, информация
  - Каждая иконка имеет уникальные параметры анимации
- **Иконки**: 
  - `StarIcon` - звезды (тема премиум/оплаты)
  - `CheckCircledIcon` - галочки (статусы)
  - `CrossCircledIcon` - запреты
  - `InfoCircledIcon` - информация

### 2. FloatingIcon компонент
- **Компонент**: `FloatingIcon` (локальный компонент)
- **Назначение**: Отображение анимированной иконки, плавающей по экрану
- **Параметры**:
  - `Icon`: компонент иконки из Radix UI
  - `index`: индекс иконки для определения параметров анимации
- **Логика**: 
  - Генерирует случайные параметры анимации на основе индекса
  - Устанавливает CSS переменные для управления анимацией
  - Создает уникальное движение для каждой иконки
- **Параметры анимации**:
  - `duration`: 8-25 секунд (зависит от группы скорости)
  - `delay`: 0-2 секунды
  - `startX/startY`: 0-100% (начальная позиция)
  - `amplitudeX/amplitudeY`: 50-120px (амплитуда движения)
  - `size`: 32-53px (размер иконки)
  - `opacity`: 0.25-0.55 (прозрачность)

### 3. Код ошибки
- **Компонент**: `Text` с классом `errorCode`
- **Назначение**: Отображение крупного кода ошибки (402)
- **Логика**: 
  - Градиентная заливка (фиолетовый purple-9)
  - Анимация пульсации
  - Адаптивный размер (120px на десктопе, 80px на мобильных)

### 4. Заголовок
- **Компонент**: `Flex` с иконкой `StarIcon` и текстом
- **Назначение**: Отображение названия ошибки с визуальным акцентом
- **Логика**: 
  - Иконка звезды с анимацией переворота (`@keyframes flip`)
  - Текст "Требуется оплата"
  - Визуально подчеркивает тему оплаты/премиум

### 5. Описание ошибки
- **Компонент**: `Text` с описанием
- **Назначение**: Информирование пользователя об ошибке
- **Логика**: 
  - Текстовое описание о необходимости оплаты
  - Центрированный текст
  - Максимальная ширина 500px для удобства чтения

### 6. Кнопки навигации
- **Компонент**: `Button` с навигацией
- **Назначение**: Предоставление способов возврата в приложение
- **Кнопки**:
  - "На главную": переход на `/workflow` (главная рабочая страница)
  - "Назад": возврат в истории браузера (`router.back()`)
- **Связи**: 
  - Использует `useRouter` из Next.js для навигации

## Функции и обработчики

### FloatingIcon.useEffect
- **Назначение**: Инициализация параметров анимации для иконки
- **Логика**: 
  - Генерирует случайные параметры анимации на основе индекса
  - Устанавливает CSS переменные для управления анимацией
  - Создает уникальные параметры для каждой иконки
- **Параметры**:
  - Разные скорости для разных иконок (3 группы: быстрая, средняя, медленная)
  - Случайные значения для delay, startX, startY, amplitudeX, amplitudeY, size, opacity

### Error402Page
- **Назначение**: Главный компонент страницы ошибки 402
- **Логика**: 
  - Отображает код ошибки, описание и кнопки навигации
  - Создает анимированный фон с 35 плавающими иконками
  - Использует `AppLayout` для обертки страницы

## Стили

- **Файл**: `error-402.module.css`
- **Основные классы**:
  - `.container` - основной контейнер страницы
  - `.background` - контейнер для плавающих иконок
  - `.content` - контентный блок с информацией об ошибке
  - `.errorCode` - стили для кода ошибки (градиент, пульсация)
  - `.title` - стили для заголовка
  - `.cardIcon` - стили для иконки звезды (анимация переворота)
  - `.floatingIcon` - стили для плавающих иконок
- **Анимации**:
  - `@keyframes float` - плавание иконок
  - `@keyframes pulse` - пульсация кода ошибки
  - `@keyframes flip` - переворот иконки звезды
- **Адаптивность**: 
  - Медиа-запросы для мобильных устройств
  - Уменьшенные размеры кода ошибки и иконок

## Особенности

- Анимированный фон с 35 плавающими иконками

## Использование

Страница отображается при:
- Истечении подписки
- Попытке доступа к платному контенту без оплаты
- Необходимости обновления подписки
- Ошибках платежной системы

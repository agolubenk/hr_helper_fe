# Документация страницы: Грейды (company-settings/grades/page.tsx)

## Общее описание

Страница управления грейдами компании. Позволяет создавать, редактировать и удалять грейды, настраивать уровни внутри грейдов.

## Компоненты страницы

### 1. GradeForm
- **Расположение**: `@/components/company-settings/GradeForm`
- **Назначение**: Форма создания/редактирования грейда
- **Пропсы**: 
  - Данные грейда (для редактирования)
  - `onSave`: обработчик сохранения
- **Логика**: 
  - Форма с полями грейда
  - Валидация обязательных полей
  - Сохранение через API
- **Связи**: Используется для создания и редактирования грейдов

### 2. Список грейдов
- **Компонент**: Таблица или список
- **Назначение**: Отображение всех грейдов компании
- **Логика**: 
  - Отображает список грейдов
  - Кнопки редактирования и удаления
  - Фильтрация и поиск

## Состояние компонента

### State переменные:
- `grades`: массив всех грейдов
- `editingGrade`: грейд для редактирования
- `showAddForm`: флаг отображения формы добавления

## Функции и обработчики

### CRUD операции:
- **Создание**: Добавление нового грейда
- **Чтение**: Загрузка списка грейдов
- **Обновление**: Редактирование существующего грейда
- **Удаление**: Удаление грейда с подтверждением

## Моковые данные

### mockGrades
Массив объектов грейдов с полями:
- `id`: уникальный идентификатор
- `name`: название грейда (Junior, Middle, Senior и т.д.)
- `levels`: массив уровней внутри грейда
- `description`: описание грейда

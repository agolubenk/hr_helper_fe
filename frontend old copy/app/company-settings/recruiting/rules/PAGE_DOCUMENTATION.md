# Документация страницы: Правила рекрутинга (company-settings/recruiting/rules/page.tsx)

## Общее описание

Страница управления правилами рекрутинга. Позволяет настраивать правила и автоматизацию процесса найма.

## Компоненты страницы

### 1. Список правил
- **Компонент**: Таблица или список
- **Назначение**: Отображение всех правил рекрутинга
- **Логика**: 
  - Отображает список правил
  - Показывает условия и действия
  - Кнопки редактирования и удаления

## Состояние компонента

### State переменные:
- `rules`: массив всех правил рекрутинга
- `editingRule`: правило для редактирования
- `showAddForm`: флаг отображения формы добавления

## Функции и обработчики

### CRUD операции:
- **Создание**: Добавление нового правила
- **Чтение**: Загрузка списка правил
- **Обновление**: Редактирование существующего правила
- **Удаление**: Удаление правила с подтверждением

## Моковые данные

### mockRules
Массив объектов правил рекрутинга с полями:
- `id`: уникальный идентификатор
- `name`: название правила
- `conditions`: условия срабатывания
- `actions`: действия при срабатывании
- `priority`: приоритет правила
- `isActive`: флаг активности

# Документация страницы: Scorecard (company-settings/scorecard/page.tsx)

## Общее описание

Страница настроек Scorecard. Позволяет настраивать шаблоны Scorecard для интервью, управлять критериями оценки и вопросами.

## Компоненты страницы

### 1. ScorecardSettings
- **Расположение**: `@/components/company-settings/ScorecardSettings`
- **Назначение**: Компонент настроек Scorecard
- **Пропсы**: Нет (использует внутреннее состояние)
- **Логика**: 
  - Отображение списка шаблонов Scorecard
  - Создание новых шаблонов
  - Редактирование существующих шаблонов
  - Удаление шаблонов
  - Настройка критериев оценки
  - Настройка вопросов для интервью
- **Связи**: Используется для управления Scorecard

## Состояние компонента

### State переменные:
- `scorecards`: массив всех шаблонов Scorecard
- `editingScorecard`: шаблон для редактирования
- `showAddForm`: флаг отображения формы добавления

## Функции и обработчики

### CRUD операции:
- **Создание**: Добавление нового шаблона Scorecard
- **Чтение**: Загрузка списка шаблонов
- **Обновление**: Редактирование существующего шаблона
- **Удаление**: Удаление шаблона с подтверждением

## Моковые данные

### mockScorecards
Массив объектов шаблонов Scorecard с полями:
- `id`: уникальный идентификатор
- `name`: название шаблона
- `criteria`: массив критериев оценки
- `questions`: массив вопросов для интервью
- `template`: шаблон Scorecard

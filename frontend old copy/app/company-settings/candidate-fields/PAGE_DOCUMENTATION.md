# Документация страницы: Поля кандидатов (company-settings/candidate-fields/page.tsx)

## Общее описание

Страница управления дополнительными полями кандидатов. Позволяет создавать, редактировать и удалять кастомные поля для кандидатов.

## Компоненты страницы

### 1. CandidateFieldsSettings
- **Расположение**: `@/components/company-settings/CandidateFieldsSettings`
- **Назначение**: Компонент настроек полей кандидатов
- **Пропсы**: Нет (использует внутреннее состояние)
- **Логика**: 
  - Отображение списка всех кастомных полей
  - Создание новых полей
  - Редактирование существующих полей
  - Удаление полей
  - Настройка типов полей
- **Связи**: Используется для управления полями кандидатов

## Состояние компонента

### State переменные:
- `fields`: массив всех кастомных полей
- `editingField`: поле для редактирования
- `showAddForm`: флаг отображения формы добавления

## Функции и обработчики

### CRUD операции:
- **Создание**: Добавление нового поля
- **Чтение**: Загрузка списка полей
- **Обновление**: Редактирование существующего поля
- **Удаление**: Удаление поля с подтверждением

## Моковые данные

### mockFields
Массив объектов полей кандидатов с полями:
- `id`: уникальный идентификатор
- `name`: название поля
- `type`: тип поля (text, number, date, select, и т.д.)
- `required`: обязательное поле или нет
- `options`: опции для полей типа select (опционально)

# Документация страницы: Детальный просмотр зарплатной вилки (vacancies/salary-ranges/[id]/page.tsx)

## Общее описание

Страница детального просмотра зарплатной вилки. Отображает полную информацию о вилке и выполняет редирект на страницу списка с параметром `?detail=`.

## Компоненты страницы

### Компонент редиректа
- **Тип**: Функциональный компонент без UI
- **Назначение**: Выполнение редиректа на страницу списка зарплатных вилок с параметром детального просмотра
- **Логика**: 
  - Читает ID вилки из URL параметра `[id]`
  - Выполняет редирект на `/vacancies/salary-ranges?detail=[id]`
  - Если ID отсутствует - редирект на `/vacancies/salary-ranges`
- **Поведение**: 
  - Не рендерит никакого контента (возвращает `null`)
  - Выполняется один раз при монтировании компонента

## Состояние компонента

### State переменные:
- Нет (компонент только выполняет редирект)

## Функции и обработчики

### useEffect для редиректа
- **Назначение**: Выполнение редиректа при загрузке компонента
- **Логика**: 
  - Если ID вилки существует - редирект на `/vacancies/salary-ranges?detail=[id]`
  - Если ID отсутствует - редирект на `/vacancies/salary-ranges`
- **Поведение**: 
  - Выполняется один раз при монтировании
  - Использует `router.replace()` (не добавляет запись в историю)
  - Кодирует ID через `encodeURIComponent()` для безопасной передачи

## URL параметры

### [id]
- **Тип**: Динамический параметр маршрута
- **Назначение**: ID зарплатной вилки для просмотра
- **Пример**: `/vacancies/salary-ranges/123` → редирект на `/vacancies/salary-ranges?detail=123`
- **Обработка**: 
  - Читается через `useParams()`
  - Передается в URL параметр `?detail=` на странице списка

## Связи с другими страницами

- `/vacancies/salary-ranges/page.tsx` - страница списка зарплатных вилок, которая обрабатывает параметр `?detail=` и открывает модальное окно детального просмотра

## Особенности

- Не рендерит контент (возвращает `null`)
- Использует `router.replace()` вместо `router.push()` (не добавляет запись в историю)
- Обеспечивает обратную совместимость со старыми ссылками
- Автоматически открывает модальное окно детального просмотра на странице списка

## Причина существования

- Обеспечивает обратную совместимость со старыми ссылками на детальную страницу зарплатной вилки
- Упрощает навигацию: вместо отдельной страницы используется модальное окно на странице списка
- Позволяет использовать глубокие ссылки на просмотр зарплатной вилки

# Документация страницы: Зарплатные вилки (vacancies/salary-ranges/page.tsx)

## Общее описание

Страница управления зарплатными вилками для вакансий. Позволяет создавать, просматривать, редактировать и управлять зарплатными вилками в разных валютах (USD, BYN, PLN, EUR).

## Компоненты страницы

### 1. SalaryRangesSearchFilters
- **Расположение**: `@/components/salary-ranges/SalaryRangesSearchFilters`
- **Назначение**: Поиск и фильтры для зарплатных вилок
- **Пропсы**:
  - `searchQuery`: текущий поисковый запрос
  - `onSearchChange`: обработчик изменения поиска
  - `selectedVacancy`: выбранная вакансия ('all' или название)
  - `onVacancyChange`: обработчик изменения вакансии
  - `selectedGrade`: выбранный грейд ('all' или грейд)
  - `onGradeChange`: обработчик изменения грейда
  - `onReset`: обработчик сброса всех фильтров
- **Логика**: 
  - Поле поиска по вакансии, грейду, ID или значению
  - Фильтр по вакансии (выпадающий список)
  - Фильтр по грейду (выпадающий список)
  - Кнопка "Поиск" (TODO: реализовать функциональность)
  - Кнопка "Сбросить" (сбрасывает все фильтры)
- **Связи**: Передает изменения фильтров в родительский компонент

### 2. SalaryRangesStats
- **Расположение**: `@/components/salary-ranges/SalaryRangesStats`
- **Назначение**: Статистика по зарплатным вилкам
- **Пропсы**:
  - `total`: общее количество вилок
  - `active`: количество активных вилок
  - `inactive`: количество неактивных вилок
  - `activeTab`: активная вкладка фильтра ('active' | 'inactive' | 'all')
  - `onTabChange`: обработчик изменения активной вкладки
  - `onListViewClick`: обработчик клика на иконку списка (переключение режима отображения)
- **Логика**: 
  - Отображает общее количество вилок
  - SegmentedControl для фильтрации по статусу (Активных, Неактивных, Все)
  - Иконка списка для переключения режима отображения
  - Адаптивные тексты для мобильных устройств
- **Связи**: Управляет фильтрацией и режимом отображения

### 3. SalaryRangeCard
- **Расположение**: `@/components/salary-ranges/SalaryRangeCard`
- **Назначение**: Карточка зарплатной вилки (режим "Карточки")
- **Пропсы**:
  - `salaryRange`: объект зарплатной вилки
  - `onClick`: обработчик клика на карточку (детальный просмотр)
  - `onToggleActive`: обработчик переключения активности вилки
- **Логика**: 
  - Заголовок: название вакансии и грейд (на цветном фоне)
  - Детали: зарплатные вилки в USD, BYN, PLN, EUR
  - Дата обновления
  - Кнопка активации/деактивации
- **Связи**: Используется только в режиме `viewMode === 'cards'`

### 4. SalaryRangeListItem
- **Расположение**: `@/components/salary-ranges/SalaryRangeListItem`
- **Назначение**: Элемент списка зарплатной вилки (режим "Список")
- **Пропсы**: Те же, что у `SalaryRangeCard`
- **Логика**: 
  - Горизонтальное расположение информации
  - Адаптивная верстка для мобильных устройств
- **Связи**: Используется только в режиме `viewMode === 'list'`

### 5. CreateSalaryRangeModal
- **Расположение**: `@/components/salary-ranges/CreateSalaryRangeModal`
- **Назначение**: Модальное окно создания новой зарплатной вилки
- **Пропсы**:
  - `isOpen`: флаг открытости
  - `onClose`: обработчик закрытия
  - `onSave`: обработчик сохранения
- **Логика**: Форма создания зарплатной вилки с полями для всех валют
- **Связи**: После сохранения обновляет список вилок

### 6. SalaryRangeDetailModal
- **Расположение**: `@/components/salary-ranges/SalaryRangeDetailModal`
- **Назначение**: Модальное окно детального просмотра/редактирования зарплатной вилки
- **Пропсы**:
  - `open`: флаг открытости
  - `onOpenChange`: обработчик изменения открытости
  - `salaryRangeId`: ID зарплатной вилки
  - `mode`: 'view' | 'edit'
- **Логика**: 
  - Режим просмотра: только чтение данных
  - Режим редактирования: возможность изменения всех полей
- **Связи**: Открывается при клике на вилку или через URL параметр `?detail=`

## Состояние компонента

### State переменные:
- `viewMode`: режим отображения ('cards' | 'list' | 'table')
- `activeTab`: активная вкладка фильтра ('active' | 'inactive' | 'all')
- `searchQuery`: поисковый запрос
- `selectedVacancy`: выбранная вакансия ('all' или название)
- `selectedGrade`: выбранный грейд ('all' или грейд)
- `isCreateModalOpen`: флаг открытия модального окна создания
- `detailModalOpen`: флаг открытия модального окна детального просмотра
- `detailModalId`: ID вилки для детального просмотра

## Функции и обработчики

### Фильтрация
- `filteredSalaryRanges`: отфильтрованный список вилок по:
  1. Поиску: вакансия, грейд, ID или значение зарплаты
  2. Вакансии: совпадение с `selectedVacancy` (или 'all')
  3. Грейду: совпадение с `selectedGrade` (или 'all')
  4. Статусу: активность (`isActive`) совпадает с `activeTab` (или 'all')

### Экспорт Excel
- **Функция**: `handleExportExcel()`
- **Логика**:

  - Использует библиотеку XLSX

## Режимы отображения

### Режим "Карточки" (viewMode === 'cards')
- **Компонент**: `SalaryRangeCard`
- **Особенности**: Вертикальное расположение, цветной заголовок

### Режим "Список" (viewMode === 'list')
- **Компонент**: `SalaryRangeListItem`
- **Особенности**: Горизонтальное расположение, адаптивная верстка

### Режим "Таблица" (viewMode === 'table')
- **Компонент**: `Table` из Radix UI
- **Особенности**: Табличное отображение всех данных

## Моковые данные

- `salaryUsd`, `salaryByn`, `salaryPln`, `salaryEur`: объекты с `min` и `max`
- `isActive`: флаг активности вилки
- `updatedAt`: дата последнего обновления (ISO строка)

## Форматы зарплат

- **USD, BYN**: отображаются как "net" (чистая зарплата)
- **PLN, EUR**: отображаются как "gross" (грязная зарплата)
- **Форматирование**: числа форматируются с разделителями тысяч через `Intl.NumberFormat`

## URL параметры

### ?detail=ID
- **Назначение**: Открытие модального окна детального просмотра вилки
- **Логика**: 
  - Читается через `useSearchParams()`
  - Если параметр существует - открывает `SalaryRangeDetailModal`
  - Используется для редиректа с `/vacancies/salary-ranges/[id]`

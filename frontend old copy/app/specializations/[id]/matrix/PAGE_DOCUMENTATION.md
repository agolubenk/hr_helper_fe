# Документация страницы: Матрица навыков (specializations/[id]/matrix/page.tsx)

## Общее описание

Страница настройки матрицы компетенций выбранной специализации. Компетенции разбиты по табам: технические навыки, поведенческие навыки, задачи, тесты. В каждом табе задаётся система оценки (шкала из настроек компании) и редактируются блоки и элементы.

## Компоненты страницы

### Табы
- **Технические навыки** — блоки с техническими компетенциями (уровень, вес, описание, темы, ссылка на ответы).
- **Поведенческие навыки** — блоки с поведенческими компетенциями (уровень, вес, описание, проходной балл, темы, отказы, ссылка на ответы).
- **Задачи** — блоки с задачами (привязка к навыкам, темы, вес в блоке).
- **Тесты** — блоки с тестами (название, описание, проходной балл, вес).

### Элементы интерфейса
- **Система оценки** — в начале каждого таба: Select из шкал (`lib/ratingScale.getRatingScales()`). Выбор сохраняется по `specId` и типу таба в localStorage.
- **Уровень / Шаблон** — уровень грейда из настроек грейдирования специализации (или `DEFAULT_GRADE_LEVELS`), шаблон блоков (Frontend/Backend/DevOps).
- **Модалки**: редактирование навыка (в т.ч. вес вручную/авто, ссылка на ответы), задачи, теста, названия блока.
- **Предупреждения** — при сумме весов &lt; 100% или &gt; 100%.

## Состояние

- Данные матрицы и дерева — из `useSpecializations()` (SpecializationsContext).
- Уровни для селектов — из `gradingBySpecId` или `DEFAULT_GRADE_LEVELS`.
- Выбранная шкала по табу — в localStorage через `getRatingScaleIdForSpec` / `setRatingScaleForSpecTab`.

## Связи с другими страницами

- `/company-settings/grades` — уровни грейдов.
- `/company-settings/rating-scales` — список шкал для выбора системы оценки.
- `/recr-chat/.../assessment/new` и `.../edit` — шкала и блоки компетенций берутся по специализации, связанной с вакансией.

## Особенности

- Скролл центральной панели с табами обеспечивается классом `centerTabsRoot` в layout матрицы.
- Для поведенческих навыков в модалке доступны только: описание, уровень, проходной, вес, темы, отказы, ссылка на ответы.

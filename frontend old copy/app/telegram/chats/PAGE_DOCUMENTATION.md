# Документация страницы: Telegram Чаты (telegram/chats/page.tsx)

## Общее описание

Страница управления чатами Telegram (`/telegram/chats`) - это самый сложный и функциональный компонент модуля Telegram интеграции, содержащий более 3900 строк кода. Страница предоставляет полный функционал для работы с чатами Telegram, включая просмотр переписки, отправку сообщений с форматированием, управление настройками чатов и аккаунта, организацию чатов в папки, управление исключениями уведомлений и многое другое.

**Путь**: `/telegram/chats`

## Компоненты страницы

### 1. Управление чатами

**Отображение списка чатов:**
- Список всех чатов с аватарами, названиями и превью последнего сообщения
- Отображение даты/времени последнего сообщения
- Счетчик непрочитанных сообщений (Badge)
- Иконка замочка для заблокированных контактов
- Подсветка выбранного чата

**Поиск и фильтрация:**
- Поиск чатов по названию в реальном времени (без учета регистра)
- Фильтрация по папкам (выбор папки показывает только чаты из этой папки)
- Переключение между типами чатов: "Чаты" / "Каналы и группы"
- Папка "Все" для просмотра всех чатов

**Выбор и просмотр чата:**
- Клик по чату открывает окно чата справа
- Автоматическое закрытие мобильного меню при выборе чата
- Сброс выбранной подгруппы при переключении чата

### 2. Просмотр и отправка сообщений

**Просмотр переписки:**
- Отображение истории сообщений в выбранном чате
- Разные стили для сообщений пользователя (справа, синий фон) и из Telegram (слева, серый фон)
- Отображение авторов сообщений в группах (кроме сообщений пользователя)
- Автоматическая прокрутка к последнему сообщению при открытии чата
- Кнопка прокрутки к последнему сообщению при скролле вверх
- Отображение прикрепленных файлов в сообщениях

**Отправка сообщений:**
- Поле ввода с поддержкой визуального форматирования (Rich Text Input)
- Прикрепление файлов к сообщениям
- Отображение списка прикрепленных файлов перед отправкой
- Отправка сообщения по Enter (Shift+Enter для новой строки)
- Очистка поля ввода после отправки

**Форматирование сообщений:**
- Поддержка Markdown-подобного форматирования:
  - **Жирный текст** (`**текст**`)
  - *Курсив* (`*текст*`)
  - __Подчеркивание__ (`__текст__`)
  - ~~Зачеркивание~~ (`~~текст~~`)
  - `Инлайн код`
  - Блоки кода с кнопкой копирования
  - Цитаты с полупрозрачным фоном и пунктирной левой границей
  - Ссылки с превью (домен, полный URL, открытие в новой вкладке)
- Заголовки (только для сообщений пользователя, не для Telegram)
- Визуальное отображение форматирования в поле ввода (не Markdown синтаксис)

**Rich Text Input:**
- Визуальное отображение форматирования (жирный текст выглядит жирным, не `**текст**`)
- Горячие клавиши:
  - `Ctrl+B` / `Cmd+B` - жирный текст
  - `Ctrl+I` / `Cmd+I` - курсив
  - `Ctrl+U` / `Cmd+U` - подчеркивание
  - `Ctrl+Shift+X` / `Cmd+Shift+X` - зачеркивание
  - `Ctrl+Shift+.` / `Cmd+Shift+.` - цитата
  - `Ctrl+K` / `Cmd+K` - вставка ссылки
- `Shift+Enter` для новой строки, `Enter` для отправки
- Тултип для вставки ссылок с возможностью редактирования текста ссылки
- Сохранение позиции курсора при форматировании
- Повторное нажатие горячей клавиши снимает форматирование

### 3. Группы и подгруппы

**Поддержка групп:**
- Группы могут содержать вложенные подгруппы (subChats)
- Скролл-меню подгрупп под заголовком чата
- Вариант "Единый чат" для просмотра всех сообщений группы сразу
- Фильтрация сообщений по выбранной подгруппе
- Отображение авторов сообщений в группах (только для сообщений от других пользователей)

**Переключение между подгруппами:**
- Горизонтальный скролл-меню с кнопками подгрупп
- Выделение активной подгруппы
- Автоматическая фильтрация сообщений при выборе подгруппы

### 4. Настройки чатов

**Модальное окно настроек чата:**
- Открывается при клике на кнопку настроек (GearIcon) или на заголовок чата
- Содержит три раздела: "Информация", "Настройки", "Действия"

**Раздел "Информация":**
- Аватар чата (кликабельный для просмотра карусели изображений)
- Кнопка загрузки изображения (только для личных чатов, не для групп/каналов)
- Редактируемые поля (инлайн редактирование):
  - Имя (клик по значению открывает поле ввода)
  - Фамилия (клик по значению открывает поле ввода)
  - Номер телефона (клик по значению открывает поле ввода)
  - Имя пользователя (клик по значению открывает поле ввода, с кнопкой копирования)
- Нередактируемые поля:
  - Никнейм
  - О себе (About)
  - Ссылка (Link)
  - Заметка (Note) - редактируется через кнопку с иконкой карандаша

**Раздел "Настройки":**
- Переключатель "Автоматические ответы" (Switch)
- Поле заметки (TextArea) с кнопкой редактирования

**Раздел "Действия":**
- Кнопка "Изменить контакт" - открывает форму редактирования всех полей
- Кнопка "Удалить контакт" - показывает подтверждение через toast
- Кнопка "Заблокировать" / "Разблокировать" - в зависимости от статуса контакта
  - Для заблокированных контактов показывается "Разблокировать" (зеленая кнопка)
  - Для незаблокированных - "Заблокировать" (красная кнопка)

**Блокировка контактов:**
- Заблокированные контакты отображаются с иконкой замочка в углу аватара
- В настройках чата кнопка меняется на "Разблокировать"
- Подтверждение блокировки/разблокировки через toast с действиями

**Удаление контактов:**
- Подтверждение удаления через toast с кнопками "Отмена" и "Удалить"
- После удаления чат исчезает из списка
- Если удаленный чат был выбран, выбор сбрасывается

**Редактирование контакта:**
- Два режима редактирования:
  1. Инлайн редактирование отдельных полей (клик по значению)
  2. Полное редактирование через кнопку "Изменить контакт"
- Сохранение изменений по Enter или при потере фокуса
- Отмена изменений по Escape

### 5. Настройки аккаунта

**Модальное окно настроек аккаунта:**
- Открывается через кнопку "Настройки аккаунта" в субменю
- Три представления: главное, "Мой аккаунт", "Уведомления и звуки"

**Главное представление:**
- Информация о профиле: аватар, имя, телефон, username
- Навигация к разделам:
  - "Мой аккаунт" - редактирование информации профиля
  - "Уведомления и звуки" - настройки уведомлений
  - "Настройки папок" - управление папками чатов

**Раздел "Мой аккаунт":**
- Аватар профиля (кликабельный для просмотра карусели)
- Кнопка загрузки изображения (CameraIcon)
- Редактируемые поля (инлайн):
  - Имя (клик по значению открывает поле ввода)
  - Имя пользователя (клик по значению открывает поле ввода)
- Нередактируемые поля:
  - Номер телефона
- Поле "О себе" (TextArea, редактируемое)
- Подсказка о назначении username

**Раздел "Уведомления и звуки":**
- Переключатель "Включить уведомления"
- Раздел "Настройки звука":
  - Переключатель "Включить звуки"
- Раздел "Уведомления из чатов":
  - "Личные чаты" - переключатель и кликабельный элемент для управления исключениями
  - "Группы" - переключатель и кликабельный элемент для управления исключениями (показывает количество исключений)
  - "Каналы" - переключатель и кликабельный элемент для управления исключениями (показывает количество исключений)

**Управление изображениями профиля:**
- Загрузка изображений через кнопку с иконкой камеры
- Карусель для просмотра всех загруженных изображений
- Установка главного изображения (кнопка "Установить как главное" в карусели)
- Отображение главного изображения в аватаре профиля

### 6. Управление папками

**Субменю папок:**
- Горизонтальный скролл-меню с папками вверху страницы
- Папка "Все" всегда первая
- Клик по папке фильтрует список чатов

**Модальное окно управления папками:**
- Открывается через кнопку "Настройки папок" в настройках аккаунта
- Заголовок "Папки" с кнопкой назад и закрытия
- Раздел "Мои папки":
  - Список всех папок с иконками
  - Отображение количества чатов в каждой папке
  - Иконка с тултипом о том, что содержимое папок определяется только в Telegram
- Функции для каждой папки (кроме "Все"):
  - Редактирование названия (кнопка с иконкой карандаша)
  - Удаление папки (кнопка с иконкой корзины)
  - Перетаскивание для изменения порядка (drag handle)
- Кнопка "Создать новую папку":
  - При клике открывает поле ввода названия
  - Кнопки "Создать" и "Отмена"

**Drag-and-Drop:**
- Используется библиотека `@dnd-kit`
- Перетаскивание папок для изменения порядка
- Папка "Все" не перетаскивается
- Визуальная обратная связь при перетаскивании (полупрозрачность)
- Сохранение нового порядка

**Создание папки:**
- Ввод названия новой папки
- Создание папки с иконкой по умолчанию
- Показ уведомления об успешном создании

**Редактирование папки:**
- Инлайн редактирование названия (клик по кнопке карандаша)
- Сохранение по Enter или при потере фокуса
- Отмена по Escape

**Удаление папки:**
- Подтверждение удаления через toast
- При удалении чаты из папки перемещаются в папку "Все"
- Если удаленная папка была выбрана, выбор сбрасывается

### 7. Исключения уведомлений

**Модальное окно исключений:**
- Открывается при клике на элемент "Личные чаты", "Группы" или "Каналы" (не на переключатель)
- Заголовок "Добавить исключение"
- Кнопка "Добавить исключение" с иконкой плюса
- Поле поиска для фильтрации чатов
- Список чатов соответствующего типа:
  - Исключенные чаты показываются первыми с подзаголовком "Без уведомлений"
  - Кнопка "Убрать" для исключенных чатов
  - Неисключенные чаты показываются ниже с подзаголовком типа чата
- Клик по чату добавляет/удаляет его из исключений

**Отображение количества исключений:**
- В подзаголовках элементов "Группы" и "Каналы" показывается количество исключений
- Формат: "Включены, N исключений" (с правильным склонением)

### 8. Адаптивность

**Десктоп:**
- Двухколоночный layout: список чатов слева, окно чата справа
- Фиксированное субменю вверху
- Полная функциональность доступна

**Мобильные устройства:**
- Список чатов скрыт по умолчанию
- Кнопка меню (HamburgerMenuIcon) в субменю для открытия списка
- Полноэкранный модальный список чатов при открытии
- Автоматическое закрытие списка при выборе чата
- Оверлей для закрытия меню при клике вне его
- Адаптивная высота контейнеров (учет высоты хедера и субменю)

## Функции и обработчики

### Управление чатами

**handleChatClick(chatId: string)**
- Выбирает чат для отображения
- Сбрасывает выбранную подгруппу
- Закрывает мобильное меню на мобильных устройствах

**handleChatSettingsClick(chatId: string, e: React.MouseEvent)**
- Открывает модальное окно настроек чата
- Загружает текущие настройки чата
- Инициализирует поля редактирования

### Управление сообщениями

**handleSendMessage()**
- Отправляет сообщение в выбранный чат
- Добавляет сообщение в массив сообщений чата
- Обновляет превью и дату последнего сообщения
- Очищает поле ввода и прикрепленные файлы
- Прокручивает к последнему сообщению

**handleMessagesScroll()**
- Отслеживает позицию скролла в контейнере сообщений
- Определяет, находится ли пользователь внизу списка
- Показывает/скрывает кнопку прокрутки вниз

**handleScrollToBottom()**
- Прокручивает контейнер сообщений к последнему сообщению
- Вызывается при клике на кнопку прокрутки вниз

### Управление файлами

**handleFileAttach()**
- Открывает диалог выбора файла
- Добавляет выбранные файлы в массив attachedFiles

**handleRemoveFile(fileId: string)**
- Удаляет файл из массива attachedFiles

### Управление контактами

**handleSaveContact()**
- Сохраняет изменения контакта в состояние chats
- Выходит из режима редактирования

**handleSaveInlineField(field: 'name' | 'lastName' | 'phone' | 'username')**
- Сохраняет изменение отдельного поля контакта (инлайн редактирование)
- Обновляет только указанное поле

**handleBlockContact()**
- Показывает подтверждение блокировки через toast
- При подтверждении вызывает performBlockContact

**performBlockContact()**
- Блокирует контакт (устанавливает isBlocked: true)
- Закрывает модальное окно
- Показывает уведомление об успешной блокировке

**handleUnblockContact()**
- Показывает подтверждение разблокировки через toast
- При подтверждении вызывает performUnblockContact

**performUnblockContact()**
- Разблокирует контакт (устанавливает isBlocked: false)
- Закрывает модальное окно
- Показывает уведомление об успешной разблокировке

**handleDeleteContact()**
- Показывает подтверждение удаления через toast
- При подтверждении вызывает performDeleteContact

**performDeleteContact()**
- Удаляет чат из массива chats
- Закрывает модальное окно
- Сбрасывает выбор, если удаленный чат был выбран
- Показывает уведомление об успешном удалении

**handleCopyUsername(username: string)**
- Копирует username в буфер обмена с символом @
- Показывает визуальную обратную связь (иконка меняется на галочку)

### Управление заметками и автоответами

**handleSaveNote()**
- Сохраняет заметку для текущего чата
- Обновляет состояние чата

**handleToggleAutoReply(enabled: boolean)**

## Вычисляемые значения

### selectedChat
- **Назначение**: Находит чат по `selectedChatId` в массиве `chats`
- **Вычисление**: `chats.find(chat => chat.id === selectedChatId)`
- **Возвращает**: `TelegramChat | undefined` - выбранный чат или undefined если чат не выбран
- **Используется для**: Отображения информации о чате в окне чата

### filteredChats
- **Назначение**: Отфильтрованный список чатов по всем критериям
- **Вычисление**: `chats.filter(chat => { ... })`
- **Логика фильтрации** (все условия должны выполняться одновременно):
  1. **Поиск** (`searchQuery`): 
     - Если `searchQuery` пустой - пропускает все чаты
     - Иначе проверяет, содержит ли название чата поисковый запрос (без учета регистра)
  2. **Фильтр по папке** (`selectedFolderId`): 
     - Если `selectedFolderId === null` (папка "Все") - пропускает все чаты
     - Иначе проверяет совпадение `chat.folderId === selectedFolderId`
  3. **Фильтр по типу** (`viewType`): 
     - Если `viewType === 'chats'` - показывает только личные чаты (`type === 'chat'`)
     - Если `viewType === 'channels'` - показывает каналы и группы (`type === 'channel' || type === 'group'`)
- **Используется для**: Отображения в списке чатов
- **useMemo**: Пересчитывается только при изменении `chats`, `searchQuery`, `selectedFolderId`, `viewType`

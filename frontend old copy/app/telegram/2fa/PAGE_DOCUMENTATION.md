# Документация страницы: Telegram 2FA (telegram/2fa/page.tsx)

## Общее описание

Страница двухфакторной аутентификации Telegram. Позволяет настроить 2FA для Telegram аккаунта.

## Компоненты страницы

### 1. Навигационные вкладки
- **Компонент**: `Link` компоненты
- **Назначение**: Переключение между разделами Telegram
- **Вкладки**:
  - Вход (текущая страница)
  - 2FA (текущая страница, активна)
  - Чаты (список чатов)
- **Логика**: 
  - Подсвечивает активную вкладку
  - При клике происходит переход на соответствующую страницу
- **Связи**: 
  - `/telegram` - страница входа
  - `/telegram/chats` - страница чатов

### 2. Форма настройки 2FA
- **Компонент**: Форма с полями
- **Назначение**: Настройка двухфакторной аутентификации
- **Поля**:
  - Пароль 2FA
  - Подтверждение пароля
  - Подсказка для восстановления

## Состояние компонента

### State переменные:
- `password`: значение поля пароля 2FA
- `confirmPassword`: значение поля подтверждения пароля
- `hint`: подсказка для восстановления
- `loading`: флаг загрузки

## Функции и обработчики

### handleSetup2FA()
- **Назначение**: Настройка 2FA для Telegram
- **Логика**: 
  1. Валидирует пароль и подтверждение
  2. Отправляет данные на сервер
  3. При успехе показывает уведомление

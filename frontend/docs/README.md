## Навигация по документам

| Документ | Назначение |
|----------|------------|
| **[APP_INVENTORY.md](./APP_INVENTORY.md)** | Полный инвентарь: модули, страницы, статусы, последовательность реализации, роадмап |

| **[MIGRATION_AUDIT.md](./MIGRATION_AUDIT.md)** | Аудит миграции настроек из fullstack |

## Быстрый старт

1. **Что реализовано, что нет?** → APP_INVENTORY.md, раздел 2
2. **В каком порядке делать?** → APP_INVENTORY.md, раздел 3 (Роадмап)
3. **Как устроено меню?** → REFERENCE.md, раздел 2
4. **Куда ведёт пункт X?** → MENU_AND_PAGES_REFERENCE.md
5. **Что перенесено из fullstack?** → MIGRATION_AUDIT.md

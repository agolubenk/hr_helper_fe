## Изменения вне изначального «плана» (но зафиксированы)

- Подключение **футера** и настроек задач в футере.
- Расширение **ThemeProvider** (auto-тема, миграция со старого ключа `theme`).
- Вынос **`quickButtonsStorage`** и сохранение кнопок с **страницы** настроек (раньше панель жила на захардкоженном списке).
- Редиректы/вкладки профиля (`profileActiveTab`, синхронизация по `localStorage`).
- Упрощение логов в **FloatingActions** (убраны отладочные `console.log` из старой fe_migration-версии).

## Исключения из текущего плана миграции

- `Sidebar`, `Header`, `Footer`, `FloatingActions` — заморожены и не дорабатываются в рамках текущего плана.
- Для `FloatingActions` зафиксирована позиция `left: 12px`.

   Профиль, быстрые кнопки, соцсети, интеграции — единый контракт с бэкендом; `quickButtonsStorage` оставить как fallback/offline.

   Vitest: `quickButtonsStorage`, редьюсеры вкладок профиля, критичные клики в `SocialLinksManager` (лимит 3 на платформу).

   FloatingActions исключён из текущего плана; проверяем только отсутствие регресса.

   Сравнение 3000 vs 3002 по ключевым страницам (профиль, ATS); исключённые shell-блоки не дорабатываем.

## Форма редактирования вики (WikiEditForm)

Изменения зафиксированы в `src/components/wiki/COMPONENTS_DOCUMENTATION.md` (раздел «Структура и UX формы», «Changelog»). Кратко:

- Убран верхний селект «Шаблон страницы»; шаблоны — только через «Добавить шаблон» в тулбаре и модалку.
- Три карточки на всю ширину: Содержание, Параметры (порядок, опубликовано, примечание), Действия (Удалить | Отмена, Сохранить).
- На мобильных справка в модалке по кнопке инфо в шапке формы.
- **Плавающие кнопки** вынесены в компонент **FloatingConfirmActions** (название без привязки к вики): красная «Удалить», капсула «Отмена»/«Сохранить»; привязка к `[data-app-layout-content]` (ResizeObserver), анимация расширения при прокрутке к карточке (`--expansion`); статические кнопки в карточке скрыты, пока видна панель.
- **Мобильный режим** определяется **шириной блока** карточки действий (≤800px), не viewport; при узком блоке карточка скрыта, панель всегда видна в компактном виде (только иконки).
- Подтверждение удаления — тост с кнопками «Отмена»/«Удалить» (useToast.showWarning).
- Прокрутка только в области контента (body без скролла в globals.css).

# Документация компонентов: finance

## Общее описание

Директория `components/finance` содержит компоненты для управления финансовыми настройками компании. Компоненты обеспечивают управление грейдами сотрудников, курсами валют НБРБ и налогами для польского злотого (PLN).

**Расположение**: `@/components/finance`

## Компоненты

### 1. GradesSection
- **Файл**: `GradesSection.tsx`
- **Назначение**: Компонент управления грейдами сотрудников для финансовых расчетов
- **Пропсы**: Нет (использует внутреннее состояние)
- **Функциональность**: 
  - Отображение списка грейдов в виде карточек
  - Добавление нового грейда из предопределенного списка
  - Редактирование названия грейда (inline)
  - Удаление грейда с подтверждением
  - Выбор грейда из списка ALL_POSSIBLE_GRADES
- **Интерфейс Grade**:
  - `id`: уникальный идентификатор (number)
  - `name`: название грейда (string)
- **Предопределенные грейды** (ALL_POSSIBLE_GRADES):
  - Trainee, Junior-, Junior, Junior+, Middle-, Middle, Middle+, Senior-, Senior, Senior+, Lead, Lead+, Head, C-Level
- **Особенности**: 
  - Можно добавить только те грейды, которых еще нет в списке
  - Inline редактирование названия
  - Информационное сообщение о создании грейдов в настройках компании
- **Используется в**: 
  - `/app/finance/page.tsx`: вкладка "Грейды"

### 2. CurrencyRatesSection
- **Файл**: `CurrencyRatesSection.tsx`
- **Назначение**: Компонент управления курсами валют НБРБ
- **Пропсы**: Нет (использует внутреннее состояние)
- **Функциональность**: 
  - Отображение курсов валют в таблице
  - Обновление курсов валют вручную (кнопка "Обновить валюты")
  - Отображение статуса курса (Сегодня, Вчера, Устарело, Базовая валюта)
  - Форматирование даты последнего обновления
  - Информационное сообщение об источнике данных
- **Интерфейс CurrencyRate**:
  - `id`: уникальный идентификатор (number)
  - `code`: код валюты (string, например 'BYN', 'USD', 'EUR', 'PLN')
  - `rate`: курс валюты (string)
  - `scale`: масштаб (number)
  - `fetched_at`: дата получения курса (string, ISO format)
  - `status_info`: информация о статусе (string, опционально)
  - `display_rate`: отображаемый курс (string, опционально)
- **Поддерживаемые валюты**:
  - BYN: Белорусский рубль (базовая валюта)
  - USD: Доллар США
  - EUR: Евро
  - PLN: Польский злотый
- **Особенности**: 
  - Автоматическое определение статуса на основе даты обновления
  - Анимация вращения иконки при обновлении (spinning)
  - Информационное сообщение об источнике данных (НБРБ)
- **Используется в**: 
  - `/app/finance/page.tsx`: вкладка "Курсы валют"

### 3. TaxesSection
- **Файл**: `TaxesSection.tsx`
- **Назначение**: Компонент управления налогами для польского злотого (PLN)
- **Пропсы**: Нет (использует внутреннее состояние)
- **Функциональность**: 
  - Отображение списка налогов в таблице
  - Добавление нового налога
  - Редактирование налога (inline)
  - Удаление налога с подтверждением
  - Переключение активности налога
  - Пример расчета (Net, Gross, Налоги)
  - Отображение количества активных налогов
- **Интерфейс PLNTax**:
  - `id`: уникальный идентификатор (number)
  - `name`: название налога (string)
  - `rate`: ставка налога в процентах (string)
  - `rate_decimal`: ставка в десятичном формате (number, опционально)
  - `is_active`: активен ли налог (boolean)
  - `created_at`: дата создания (string, ISO format)
  - `updated_at`: дата обновления (string, ISO format)
- **Особенности**: 
  - Валидация ставки (0-100%)
  - Автоматический расчет rate_decimal из rate
  - Inline редактирование всех полей
  - Пример расчета на основе активных налогов
  - Переключение активности через Switch
- **Используется в**: 
  - `/app/finance/page.tsx`: вкладка "Налоги PLN"

## Интерфейсы и типы

### Grade (GradesSection)
```typescript
interface Grade {
  id: number              // Уникальный идентификатор
  name: string           // Название грейда

```

### CurrencyRate (CurrencyRatesSection)
```typescript
interface CurrencyRate {
  id: number             // Уникальный идентификатор
  code: string           // Код валюты (BYN, USD, EUR, PLN)
  rate: string           // Курс валюты
  scale: number          // Масштаб
  fetched_at: string     // Дата получения курса (ISO format)
  status_info?: string   // Информация о статусе
  display_rate?: string  // Отображаемый курс
}
```

### PLNTax (TaxesSection)
```typescript
interface PLNTax {
  id: number             // Уникальный идентификатор
  name: string           // Название налога
  rate: string           // Ставка налога в процентах
  rate_decimal?: number  // Ставка в десятичном формате
  is_active: boolean     // Активен ли налог
  created_at: string     // Дата создания (ISO format)
  updated_at: string     // Дата обновления (ISO format)
}
```

## Функции и обработчики

### GradesSection

#### handleAdd
- **Назначение**: Добавление нового грейда
- **Логика**: 
  - Валидация выбранного грейда
  - Создание нового грейда с уникальным ID
  - Добавление в массив grades
  - Очистка формы

#### handleEdit
- **Назначение**: Начало редактирования грейда
- **Параметры**: `grade: Grade`
- **Логика**: Устанавливает editingId и editingName

#### handleSaveEdit
- **Назначение**: Сохранение изменений грейда
- **Логика**: 
  - Валидация названия
  - Обновление грейда в массиве
  - Очистка состояния редактирования

#### handleDelete
- **Назначение**: Удаление грейда
- **Параметры**: `id: number`
- **Логика**: Показывает Toast с подтверждением, при подтверждении удаляет грейд

### CurrencyRatesSection

#### handleUpdateRates
- **Назначение**: Обновление курсов валют
- **Логика**: 
  - Устанавливает updating в true
  - Имитация обновления (задержка 1 секунда)
  - Обновляет fetched_at и status_info
  - Показывает alert с подтверждением
  - TODO: Реализовать реальный API вызов

#### formatDate
- **Назначение**: Форматирование даты для отображения
- **Параметры**: `dateString: string`
- **Возвращает**: отформатированная строка (DD.MM.YYYY HH:MM)
- **Логика**: Использует toLocaleDateString с русской локалью

#### getStatusBadge
- **Назначение**: Определение статуса курса валюты
- **Параметры**: `rate: CurrencyRate`
- **Возвращает**: Badge компонент с цветом и текстом
- **Логика**: 
  - BYN: всегда "Базовая валюта" (синий)
  - Остальные: вычисляет разницу в часах
  - < 24 часов: "Сегодня" (зеленый)
  - < 48 часов: "Вчера" (оранжевый)
  - >= 48 часов: "Устарело" (красный)

### TaxesSection

#### handleAdd
- **Назначение**: Добавление нового налога
- **Логика**: 
  - Валидация полей (название и ставка)
  - Валидация ставки (0-100%)
  - Создание нового налога с расчетом rate_decimal
  - Добавление в массив taxes
  - Очистка формы

#### handleEdit
- **Назначение**: Начало редактирования налога
- **Параметры**: `tax: PLNTax`
- **Логика**: Устанавливает editingId и editingTax

#### handleSaveEdit
- **Назначение**: Сохранение изменений налога
- **Логика**: 
  - Валидация полей
  - Валидация ставки
  - Обновление налога с расчетом rate_decimal
  - Обновление updated_at
  - Очистка состояния редактирования

#### handleToggleActive
- **Назначение**: Переключение активности налога
- **Параметры**: `tax: PLNTax`
- **Логика**: Инвертирует is_active и обновляет updated_at

#### handleDelete
- **Назначение**: Удаление налога
- **Параметры**: `id: number`
- **Логика**: Показывает Toast с подтверждением, при подтверждении удаляет налог

## Вычисляемые значения

### GradesSection

#### availableGrades
- **Назначение**: Доступные грейды для добавления
- **Вычисление**: `ALL_POSSIBLE_GRADES.filter(gradeName => !grades.some(g => g.name === gradeName))`
- **Логика**: Фильтрует грейды, которых еще нет в списке

### TaxesSection

#### activeTaxes
- **Назначение**: Массив активных налогов
- **Вычисление**: `taxes.filter(t => t.is_active)`

#### totalRate
- **Назначение**: Общая ставка всех активных налогов
- **Вычисление**: `activeTaxes.reduce((sum, tax) => sum + parseFloat(tax.rate), 0)`

#### exampleNet, exampleGross, exampleTaxes
- **Назначение**: Пример расчета налогов
- **Вычисление**: 
  - `exampleNet = 5000` (фиксированное значение)
  - `exampleGross = exampleNet / (1 - totalRate / 100)` (если есть активные налоги)
  - `exampleTaxes = exampleGross - exampleNet`

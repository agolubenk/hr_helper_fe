# Команды и метки для Workflow Chat

## Описание команд

### #add
- **Команда:** `/add`
- **Описание:** Автоматически добавляется при прикреплении файла
- **Использование:** Только с прикрепленным файлом
- **Цвет фона:** Синий (rgba(59, 130, 246, 0.75))

### #hr_screening
- **Команда:** `/s`
- **Описание:** Используется вместе с ссылкой и сообщением пользователя
- **Использование:** Ссылка + текст сообщения
- **Цвет фона:** Зеленый (rgba(34, 197, 94, 0.75))

### #tech_screening
- **Команда:** `/t`
- **Описание:** Используется только вместе с ссылкой и сообщением пользователя
- **Использование:** Ссылка + текст сообщения
- **Цвет фона:** Оранжевый (rgba(249, 115, 22, 0.75))

### #interview
- **Команда:** `/in`
- **Описание:** Используется только вместе с ссылкой и сообщением пользователя
- **Использование:** Ссылка + текст сообщения
- **Цвет фона:** Индиго (rgba(99, 102, 241, 0.75))

### #delete
- **Команда:** `/del`
- **Описание:** Команда удаления последнего действия
- **Использование:** Только команда (без сообщения)
- **Цвет фона:** Красный (rgba(239, 68, 68, 0.75))

## Правила использования

1. Команды через слеш (`/add`, `/s`, `/t`, `/in`, `/del`) могут вводиться вручную в textarea или подтягиваться автоматически
2. После ввода команды она автоматически затирается из textarea
3. Ввод новой команды заменяет предыдущую
4. Команда `#add` автоматически добавляется при прикреплении файла
5. Команды отображаются в правом верхнем углу textarea (как метки сообщений пользователя)

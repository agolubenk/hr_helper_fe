# Документация компонента: ProfileEditForm

## Общее описание

Компонент формы редактирования профиля пользователя. Позволяет редактировать личные данные, настраивать рабочее время и управлять контактами.

**Расположение**: `components/profile/ProfileEditForm.tsx`  
**Использование**: Вкладка "Редактирование" страницы профиля (`/account/profile?tab=edit`)

## Функциональность

### Редактируемые поля

1. **Имя** (`firstName`)
   - Тип: текстовое поле
   - Статус: отключено (disabled) - редактируется только администратором
   - Валидация: обязательное поле

2. **Фамилия** (`lastName`)
   - Тип: текстовое поле
   - Статус: отключено (disabled) - редактируется только администратором
   - Валидация: обязательное поле

3. **Email** (`email`)
   - Тип: email поле
   - Статус: отключено (disabled) - редактируется только администратором
   - Валидация: обязательное поле, формат email

4. **Telegram** (`telegram`)
   - Тип: текстовое поле с префиксом @
   - Статус: редактируемое
   - Особенности: кастомная реализация с плавающей меткой

5. **LinkedIn** (`linkedin`)
   - Тип: текстовое поле
   - Статус: редактируемое
   - Особенности: кастомная реализация с плавающей меткой и SVG иконкой

6. **Рабочее время**
   - Режим 1: Все дни одинаково (`workStartTime`, `workEndTime`)
   - Режим 2: Каждый день отдельно (`workTimeByDay`)
   - Переключение через Switch "Все дни недели одинаково"

7. **Интервал между встречами** (`meetingInterval`)
   - Тип: числовое поле
   - Формат: минуты (кратно 5, от 0 до 60)

## Состояние компонента

### State переменные

- `formData`: данные формы (синхронизируется с `initialData`)
- `telegramFocused`: флаг фокуса на поле Telegram (для анимации метки)
- `linkedinFocused`: флаг фокуса на поле LinkedIn (для анимации метки)
- `isAllDaysSame`: режим рабочего времени (true = все дни одинаково, false = каждый день отдельно)
- `workTimeByDay`: рабочее время по дням недели (используется когда `isAllDaysSame === false`)

## Функции и обработчики

### handleToggleChange(checked: boolean)

**Назначение**: Переключение режима рабочего времени

**Логика синхронизации**:
- При переключении на "все дни одинаково": копирует время из первого дня (monday) в общие поля
- При переключении на "каждый день отдельно": копирует общее время во все дни недели

### handleChange(field)

**Назначение**: Фабрика обработчиков изменения полей формы

**Использование**: `onChange={handleChange('telegram')}`

### handleSubmit(e: React.FormEvent)

**Назначение**: Обработчик отправки формы

**Подготовка данных**:
- Если `isAllDaysSame === true`: включает `workStartTime` и `workEndTime`, устанавливает `workTimeByDay` в `undefined`
- Если `isAllDaysSame === false`: включает `workTimeByDay`, устанавливает `workStartTime` и `workEndTime` в `undefined`

### handleWorkTimeByDayChange(day, field, value)

**Назначение**: Изменение рабочего времени для конкретного дня

**Параметры**:
- `day`: день недели (monday, tuesday, wednesday, thursday, friday)
- `field`: поле для обновления ('start' | 'end')
- `value`: новое значение времени (формат HH:mm)

## Интерфейсы

### WorkTimeByDay

```typescript
interface WorkTimeByDay {
  monday?: { start: string; end: string }
  tuesday?: { start: string; end: string }
  wednesday?: { start: string; end: string }
  thursday?: { start: string; end: string }
  friday?: { start: string; end: string }
}
```

### ProfileEditFormProps

```typescript
interface ProfileEditFormProps {
  initialData: {
    firstName: string
    lastName: string
    email: string
    telegram?: string
    linkedin?: string
    workStartTime?: string
    workEndTime?: string
    workTimeByDay?: WorkTimeByDay
    meetingInterval?: string
  }
  onCancel: () => void
  onSave: (data: ProfileEditFormProps['initialData']) => void
}
```

## Особенности

### Кастомные поля

**Telegram**:
- Префикс @ перед полем ввода
- Плавающая метка с анимацией
- Иконка самолетика слева

**LinkedIn**:
- SVG иконка LinkedIn слева
- Плавающая метка с анимацией

### Режимы рабочего времени

**Режим 1: Все дни одинаково**
- Два поля: начало и конец рабочего времени
- Применяется ко всем рабочим дням недели

**Режим 2: Каждый день отдельно**
- Поля для каждого дня недели (понедельник - пятница)
- Каждый день имеет свое время начала и окончания

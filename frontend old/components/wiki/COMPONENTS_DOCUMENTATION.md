# Документация компонентов: wiki

## Общее описание

Директория `components/wiki` содержит компоненты для системы вики (базы знаний) в HR Helper. Компоненты обеспечивают отображение списка страниц вики, создание и редактирование страниц, просмотр детальной информации, управление тегами и категориями, загрузку файлов и историю изменений.

**Расположение**: `@/components/wiki`

## Компоненты

### Основные компоненты списка

#### 1. WikiHeader
- **Файл**: `WikiHeader.tsx`
- **Назначение**: Заголовок страницы списка вики
- **Пропсы**: нет
- **Функциональность**: 
  - Отображает заголовок "Список страниц вики"
  - Кнопки быстрых действий:
    - "Быстрый старт" (RocketIcon)
    - "Путеводитель" (PersonIcon)
    - "Создать страницу" (PlusIcon) - переход к `/wiki/new/edit`
- **Используется в**: `/app/wiki/page.tsx`

#### 2. WikiFilters
- **Файл**: `WikiFilters.tsx`
- **Назначение**: Компонент поиска и фильтров для вики
- **Пропсы**:
  - `tags`: массив тегов (Tag[])
  - `selectedTag`: выбранный тег (string | null)
  - `onTagSelect`: обработчик выбора тега ((tagId: string | null) => void)
  - `searchQuery`: текущий поисковый запрос (string)
  - `onSearchChange`: обработчик изменения поискового запроса ((query: string) => void)
- **Функциональность**: 
  - Поле поиска: поиск по названию или содержимому
  - Фильтр по тегам: выбор тега из списка (или "Все теги")
  - Цветовая индикация выбранных тегов
- **Используется в**: `/app/wiki/page.tsx`

#### 3. WikiCategory
- **Файл**: `WikiCategory.tsx`
- **Назначение**: Компонент категории страниц вики
- **Пропсы**:
  - `category`: название категории (string)
  - `pages`: массив страниц в категории (WikiPage[])
- **Функциональность**: 
  - Отображает заголовок категории с иконкой папки
  - Отображает карточки страниц в категории через WikiCard
  - Сетка карточек с адаптивной версткой
- **Используется в**: `/app/wiki/page.tsx`

#### 4. WikiCard
- **Файл**: `WikiCard.tsx`
- **Назначение**: Компонент карточки страницы вики
- **Пропсы**:
  - `page`: данные страницы вики (WikiPage)
- **Функциональность**: 
  - Отображает карточку страницы вики
  - Заголовок, теги, описание, дата
  - Иконка открытия в новой вкладке
  - Выделение первой страницы в категории "Архитектура"
  - Цветовая индикация тегов
- **Используется в**: `WikiCategory.tsx`

### Компоненты детального просмотра

#### 5. WikiDetailHeader
- **Файл**: `WikiDetailHeader.tsx`
- **Назначение**: Заголовок страницы детального просмотра вики
- **Пропсы**:
  - `title`: название страницы (string)
  - `category`: категория страницы (string)
  - `tags`: массив тегов (string[])
- **Функциональность**: 
  - Отображает название страницы, категорию и теги
  - Навигационные хлебные крошки
  - Цветовая индикация тегов
- **Используется в**: `/app/wiki/[id]/page.tsx`

#### 6. WikiDetailContent
- **Файл**: `WikiDetailContent.tsx`
- **Назначение**: Компонент содержимого страницы вики
- **Пропсы**:
  - `content`: структурированное содержимое страницы
    - `heading`: главный заголовок (string)
    - `sections`: массив секций (ContentSection[])
- **Функциональность**: 
  - Отображает структурированное содержимое страницы
  - Главный заголовок, секции, подсекции
  - Списки элементов
  - ID для навигации по секциям
- **Используется в**: `/app/wiki/[id]/page.tsx`

#### 7. WikiDetailSidebar
- **Файл**: `WikiDetailSidebar.tsx`
- **Назначение**: Боковая панель с содержанием страницы вики
- **Пропсы**:
  - `pageId`: ID страницы (string)
  - `sections`: массив секций для навигации (Array<{ title: string }>)
- **Функциональность**: 
  - Отображает содержание страницы
  - Навигация по секциям при клике
  - Плавная прокрутка к секциям
- **Используется в**: `/app/wiki/[id]/page.tsx`

#### 8. WikiDetailHistory
- **Файл**: `WikiDetailHistory.tsx`
- **Назначение**: Компонент истории изменений страницы вики
- **Пропсы**: данные истории
- **Функциональность**: 
  - Отображает историю изменений страницы
  - Версии, даты, авторы, изменения
  - Просмотр полного текста версии
- **Используется в**: `/app/wiki/[id]/page.tsx`

### Компоненты редактирования

#### 9. WikiEditForm
- **Файл**: `WikiEditForm.tsx`
- **Назначение**: Форма редактирования/создания страницы вики
- **Пропсы**:
  - `initialData`: начальные данные страницы (Partial<WikiPageData>, опционально)
  - `isNew`: флаг создания новой страницы (boolean, по умолчанию false)
- **Функциональность**: 
  - Создание новой страницы вики
  - Редактирование существующей страницы
  - Поля: название, slug, категория, связанное приложение, теги, описание, содержимое, порядок, публикация
  - Автогенерация slug из названия
  - Управление тегами через WikiTagSelector
  - Загрузка файлов через WikiFileUploadModal
  - Удаление страницы через WikiDeleteConfirmDialog
  - Примечание к изменениям
- **Используется в**: `/app/wiki/new/edit/page.tsx`, `/app/wiki/[id]/edit/page.tsx`

#### 10. WikiTagSelector
- **Файл**: `WikiTagSelector.tsx`
- **Назначение**: Компонент выбора тегов для страницы вики
- **Пропсы**: 
  - `selectedTags`: массив выбранных тегов (string[])
  - `onChange`: обработчик изменения тегов ((tags: string[]) => void)
- **Функциональность**: 
  - Выбор тегов из доступного списка
  - Отображение выбранных тегов
  - Удаление тегов
  - Цветовая индикация тегов
- **Используется в**: `WikiEditForm.tsx`

#### 11. WikiFileUploadModal
- **Файл**: `WikiFileUploadModal.tsx`
- **Назначение**: Модальное окно загрузки файлов для страницы вики
- **Пропсы**:
  - `isOpen`: флаг открытости модального окна (boolean)
  - `onClose`: обработчик закрытия (() => void)
  - `onFileUpload`: обработчик загрузки файла ((file: File) => void)
- **Функциональность**: 
  - Загрузка файлов (изображения, документы)
  - Предпросмотр загруженных файлов
  - Валидация типов файлов
- **Используется в**: `WikiEditForm.tsx`

#### 12. WikiDeleteConfirmDialog
- **Файл**: `WikiDeleteConfirmDialog.tsx`
- **Назначение**: Диалог подтверждения удаления страницы вики
- **Пропсы**:
  - `isOpen`: флаг открытости диалога (boolean)
  - `onClose`: обработчик закрытия (() => void)
  - `onConfirm`: обработчик подтверждения удаления (() => void)
- **Функциональность**: 
  - Подтверждение удаления страницы
  - Предупреждение о необратимости действия
- **Используется в**: `WikiEditForm.tsx`

## Интерфейсы и типы

### WikiPage
```typescript
interface WikiPage {
  id: string                                    // Уникальный идентификатор страницы
  title: string                                 // Название страницы
  description: string                            // Описание страницы
  tags: string[]                                // Массив тегов
  date: string                                  // Дата создания/обновления
  category: string                              // Категория страницы
}
```

### WikiPageData
```typescript
interface WikiPageData {
  id: string                                    // Уникальный идентификатор страницы
  title: string                                 // Название страницы
  slug: string                                  // URL-слаг страницы
  category: string                              // Категория страницы
  relatedApp: string                            // Связанное приложение
  tags: string[]                                // Массив тегов
  description: string                            // Описание страницы
  content: string                                // Содержимое страницы
  order: number                                 // Порядок сортировки
  isPublished: boolean                          // Флаг публикации
}
```

### ContentSection
```typescript
interface ContentSection {
  title: string                                 // Заголовок секции
  content?: string                              // Текст содержимого
  items?: string[]                              // Список элементов
  subsections?: Array<{                          // Подсекции
    title: string
    items: string[]
  }>
}
```

### Tag
```typescript
interface Tag {
  id: string                                    // Уникальный идентификатор тега
  label: string                                 // Название тега
  color: string                                 // Цвет тега (hex)
}
```

## Функции и обработчики

### WikiEditForm

#### handleChange
- **Назначение**: Обработчик изменения полей формы
- **Параметры**: `field: keyof WikiPageData`, `value: any`
- **Логика**: Обновляет значение поля в форме

#### handleSubmit
- **Назначение**: Обработчик отправки формы
- **Логика**: 
  - Валидирует обязательные поля
  - Сохраняет страницу через API
  - Перенаправляет на страницу просмотра

### WikiDetailSidebar

#### handleSectionClick
- **Назначение**: Обработчик клика на секцию в содержании
- **Параметры**: `sectionTitle: string`
- **Логика**: 
  - Находит элемент секции по ID
  - Прокручивает к секции с плавной анимацией

## Константы

### AVAILABLE_TAGS
- **Тип**: массив объектов с полями `id`, `label`, `color`
- **Назначение**: Доступные теги для страниц вики
- **Используется в**: `WikiTagSelector.tsx`

### tagColors
- **Тип**: `Record<string, string>`
- **Назначение**: Цвета для тегов (соответствуют цветам из фильтров)
- **Используется в**: `WikiCard.tsx`

## Особенности реализации

1. **Структурированное содержимое**: WikiDetailContent поддерживает секции, подсекции и списки
2. **Автогенерация slug**: Slug автоматически генерируется из названия при создании
3. **Цветовая индикация тегов**: Каждый тег имеет свой цвет для визуального различия
4. **Навигация по секциям**: WikiDetailSidebar обеспечивает быструю навигацию по содержимому
5. **История изменений**: Отслеживание всех изменений страницы с версиями
6. **Загрузка файлов**: Поддержка загрузки изображений и документов
7. **Фильтрация по тегам**: Удобная фильтрация страниц по тегам
8. **Поиск**: Поиск по названию и содержимому страниц
9. **Категории**: Организация страниц по категориям
10. **Публикация**: Управление публикацией страниц (опубликовано/не опубликовано)

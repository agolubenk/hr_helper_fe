## 📁 Структура документации

Каждая страница имеет свой документ `PAGE_DOCUMENTATION.md` в папке страницы, содержащий:

1. **Общее описание** - назначение и функциональность
2. **Компоненты страницы** - описание всех компонентов
3. **Состояние компонента** - state переменные
4. **Функции и обработчики** - описание логики
5. **Фильтрация и поиск** - если применимо
6. **Моковые данные** - структура данных
7. **TODO: Замена моковых данных на API** - список задач
8. **Стили** - используемые CSS модули
9. **Связи с другими страницами** - навигация
10. **Особенности** - уникальные особенности

# Документация страницы: Превью специализации (specializations/[id]/preview/page.tsx)

## Общее описание

Страница превью выбранной специализации: сводное отображение информации о специализации без бейджей с количеством сотрудников и вакансий.

## Компоненты страницы

- Контент превью рендерится в основной области layout.
- **PreviewSidebar** в layout отображает краткое превью в боковой панели (без бейджей «N сотрудников», «N вакансий»).

## Связи

- Переход из дерева специализаций или из матрицы/инфо/грейдов/карьеры/аллокации/вакансий по той же специализации.
- Данные — из контекста специализаций (`useSpecializations`).

## Особенности

- Бейджи с числом сотрудников и вакансий в превью убраны по требованию.

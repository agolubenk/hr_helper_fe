# Документация раздела: Специализации (specializations)

## Общее описание

Раздел для управления деревом специализаций компании: просмотр дерева, настройка матрицы компетенций, превью, грейды, карьера, аллокация, вакансии по специализации.

## Маршруты

| Маршрут | Описание |
|---------|----------|
| `/specializations` | Список/дерево специализаций (главная страница раздела) |
| `/specializations/[id]` | Обзор специализации (редирект или контент по умолчанию) |
| `/specializations/[id]/matrix` | Матрица навыков (технические, поведенческие, задачи, тесты) |
| `/specializations/[id]/preview` | Превью специализации |
| `/specializations/[id]/info` | Информация о специализации |
| `/specializations/[id]/grades` | Грейдирование специализации |
| `/specializations/[id]/career` | Карьерная лестница |
| `/specializations/[id]/allocation` | Аллокация |
| `/specializations/[id]/vacancies` | Вакансии по специализации |

## Компоненты раздела

- **TreeSidebar** — дерево специализаций в боковой панели (без бейджей «N вак · N сот · N ур.»).
- **PreviewSidebar** — превью выбранной специализации в боковой панели.
- **SpecializationsContext** — контекст с деревом и данными грейдирования (gradingBySpecId, уровни по специализации).
- **layout.tsx** — общий layout с сайдбарами и областью контента; в матрице используется класс для корректного скролла центральной панели.

## Связи

- Настройки грейдов (`/company-settings/grades`) — уровни в матрице и в настройках специализации.
- Шкалы оценок (`/company-settings/rating-scales`) — выбор системы оценки в табах матрицы.
- Оценки кандидатов (`/recr-chat/.../assessment/new`, `edit`) — шкала и блоки компетенций привязаны к специализации вакансии.

## 📖 Обзор модуля

### Назначение

Модуль `errors/` содержит все страницы ошибок приложения:
- ✅ HTTP ошибки (401, 402, 403, 404, 500)
- ✅ Визуально привлекательное отображение ошибок
- ✅ Навигация обратно в приложение
- ✅ Единый дизайн и функциональность

### Структура

```
errors/
├── 401/                    # Требуется авторизация
│   ├── page.tsx
│   ├── error-401.module.css
│   └── PAGE_DOCUMENTATION.md
├── 402/                    # Требуется оплата
│   ├── page.tsx
│   ├── error-402.module.css
│   └── PAGE_DOCUMENTATION.md
├── 403/                    # Доступ запрещен
│   ├── page.tsx (forbidden/)
│   ├── forbidden.module.css
│   └── PAGE_DOCUMENTATION.md
├── 404/                    # Страница не найдена
│   ├── page.tsx
│   ├── not-found.module.css
│   └── PAGE_DOCUMENTATION.md
├── 500/                    # Ошибка сервера
│   ├── page.tsx
│   ├── error-500.module.css
│   └── PAGE_DOCUMENTATION.md
├── PAGE_DOCUMENTATION.md
└── ERROR_PAGES_COMPLETE_DOCUMENTATION.md
```

## 🧩 Общие компоненты

### FloatingIcon

**Назначение:** Анимированная иконка, плавающая по экрану

**Параметры анимации:**
- `duration`: 8-25 секунд (зависит от группы скорости)
- `delay`: 0-2 секунды
- `startX/startY`: 0-100% (начальная позиция)
- `amplitudeX/amplitudeY`: 50-120px (амплитуда движения)
- `size`: 32-53px (размер иконки)
- `opacity`: 0.25-0.55 (прозрачность)

**Группы скоростей:**
- Быстрая: 8-12 секунд (индексы 0, 3, 6, ...)
- Средняя: 12-18 секунд (индексы 1, 4, 7, ...)
- Медленная: 18-25 секунд (индексы 2, 5, 8, ...)

**Реализация:**
```typescript
function FloatingIcon({ Icon, index }: FloatingIconProps) {
  const iconRef = useRef<HTMLDivElement>(null)
  
  useEffect(() => {
    // Генерация случайных параметров
    // Установка CSS переменных
  }, [index])
  
  return <div ref={iconRef} className={styles.floatingIcon}>
    <Icon width={32} height={32} />
  </div>
}
```

## 🎨 Анимации и стили

### CSS Анимации

#### @keyframes float
Плавающее движение иконок:
```css
@keyframes float {
  0%, 100% { transform: translate(0, 0) rotate(0deg) scale(1); }
  25% { transform: translate(var(--amplitude-x), -var(--amplitude-y)) rotate(90deg) scale(1.1); }
  50% { transform: translate(0, -var(--amplitude-y) * 2) rotate(180deg) scale(0.9); }
  75% { transform: translate(-var(--amplitude-x), -var(--amplitude-y)) rotate(270deg) scale(1.1); }
}
```

#### @keyframes pulse
Пульсация кода ошибки:
```css
@keyframes pulse {
  0%, 100% { transform: scale(1); opacity: 1; }
  50% { transform: scale(1.05); opacity: 0.9; }
}
```

#### Специфичные анимации
- **401**: `@keyframes bounce` - подпрыгивание иконки пользователя
- **402**: `@keyframes flip` - переворот иконки звезды
- **403**: `@keyframes shake` - тряска иконки замка
- **404**: `@keyframes searchRotate` и `@keyframes searchWrapper` - вращение и покачивание поиска
- **500**: `@keyframes shake` - тряска иконки предупреждения

### CSS Переменные

Управление анимацией через CSS переменные:
```css
.floatingIcon {
  --duration: 15s;
  --delay: 0.5s;
  --start-x: 50%;
  --start-y: 50%;
  --amplitude-x: 80px;
  --amplitude-y: 80px;
  --size: 40px;
  --opacity: 0.4;
}
```

### Адаптивность

**Мобильные устройства:**
- Уменьшенный размер кода ошибки: 120px → 80px
- Уменьшенный размер иконок: 32px → 24px
- Уменьшенные отступы контентного блока
- Оптимизированные размеры шрифтов

**Медиа-запросы:**
```css
@media (max-width: 768px) {
  .errorCode { font-size: 80px; }
  .floatingIcon { width: 24px; height: 24px; }
}
```

## ⚡ Производительность

### Оптимизации

1. **CSS анимации вместо JavaScript**
   - Использование `@keyframes` для анимации
   - GPU ускорение через `transform` и `opacity`

2. **CSS переменные**
   - Параметры передаются через CSS переменные
   - Минимальные перерисовки DOM

3. **will-change**
   - Указание браузеру об оптимизации:
   ```css
   .floatingIcon {
     will-change: transform, opacity;
   }
   ```

4. **Количество иконок**
   - 35-40 иконок оптимально для производительности
   - Больше иконок может замедлить анимацию

### Рекомендации

- ✅ Использовать CSS анимации
- ✅ Минимизировать количество DOM элементов
- ✅ Использовать `will-change` для анимируемых элементов
- ✅ Оптимизировать размеры иконок
- ✅ Тестировать на слабых устройствах

## 🔗 Связи с другими модулями

### AppLayout
Все страницы ошибок обернуты в `AppLayout`:
- Единообразный layout
- Заголовок страницы в браузере
- Навигация и меню

### useRouter
Используется для навигации:
- `router.push('/workflow')` - главная страница
- `router.push('/account/login')` - страница входа (401)
- `router.back()` - возврат в истории

### Автоматическое использование
- **404**: Next.js автоматически использует `app/not-found.tsx`
- **500**: Может быть использована при ошибках сервера
- **401/403**: Используются при ошибках авторизации

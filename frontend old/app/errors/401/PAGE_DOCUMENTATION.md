# Документация страницы: Ошибка 401 (error-401/page.tsx)

## Общее описание

Страница ошибки 401 (Unauthorized). Отображается при отсутствии авторизации или истечении сессии.

## Компоненты страницы

### 1. Сообщение об ошибке
- **Компонент**: `Text` с описанием
- **Назначение**: Информирование пользователя об ошибке
- **Логика**: 
  - Отображает код ошибки 401
  - Показывает сообщение "Не авторизован"
  - Описание проблемы

### 2. Кнопка перехода на страницу входа
- **Компонент**: `Button` с навигацией
- **Назначение**: Переход на страницу входа
- **Логика**: 
  - При клике происходит переход на `/account/login`
  - Позволяет пользователю авторизоваться
- **Связи**: Навигация на `/account/login`

# Документация страницы: Финансы (finance/page.tsx)

## Общее описание

Страница финансовых настроек компании. Управление грейдами сотрудников, курсами валют и налоговыми настройками для расчета зарплатных вилок и финансовых метрик.

**Путь**: `/finance`

## Компоненты страницы

### 1. FinancePage (основной компонент)
- **Расположение**: `@/app/finance/page.tsx`
- **Назначение**: Главный компонент страницы с оберткой Suspense
- **Функциональность**: 
  - Оборачивает FinancePageContent в Suspense для корректной работы с useSearchParams()
  - Показывает fallback (загрузку) пока компонент загружается
- **Связи**: 
  - FinancePageContent: основной компонент содержимого

### 2. FinancePageContent (основной компонент содержимого)
- **Расположение**: `@/app/finance/page.tsx`
- **Назначение**: Основной компонент содержимого страницы финансов
- **Функциональность**: 
  - Управление активной вкладкой
  - Синхронизация активной вкладки с URL параметром ?tab=
  - Отображение вкладок и их содержимого
- **Связи**: 
  - GradesSection: вкладка "Грейды"
  - CurrencyRatesSection: вкладка "Курсы валют"
  - TaxesSection: вкладка "Налоги PLN"

### 3. GradesSection
- **Расположение**: `@/components/finance/GradesSection.tsx`
- **Назначение**: Настройка грейдов и уровней сотрудников

- **Функциональность**: 
  - Отображение списка грейдов в виде карточек
  - Добавление новых грейдов из предопределенного списка
  - Редактирование существующих грейдов (inline)

  - Выбор грейдов из списка возможных (Trainee, Junior-, Junior, Junior+, Middle-, Middle, Middle+, Senior-, Senior, Senior+, Lead, Lead+, Head, C-Level)
- **Связи**: 
  - useToast: для отображения уведомлений при удалении
  - API: загрузка и сохранение грейдов (TODO)

### 4. CurrencyRatesSection
- **Расположение**: `@/components/finance/CurrencyRatesSection.tsx`

## Интерфейсы и типы

### Grade
```typescript
interface Grade {
  id: number                    // Уникальный идентификатор грейда
  name: string                  // Название грейда (Trainee, Junior, Middle, Senior и т.д.)
}
```

### CurrencyRate
```typescript
interface CurrencyRate {
  id: number                    // Уникальный идентификатор курса
  code: string                  // Код валюты (BYN, USD, EUR, PLN)
  rate: string                  // Курс валюты (строка для точности)
  scale: number                 // Масштаб (количество единиц валюты)
  fetched_at: string            // Дата и время получения курса (ISO строка)
  status_info?: string          // Информация о статусе (опционально)
  display_rate?: string         // Отображаемый курс (опционально)
}
```

### PLNTax
```typescript
interface PLNTax {
  id: number                    // Уникальный идентификатор налога
  name: string                  // Название налога
  rate: string                  // Ставка налога в процентах (строка)
  rate_decimal?: number         // Ставка налога в десятичном виде (опционально)
  is_active: boolean            // Флаг активности налога
  created_at: string            // Дата создания (ISO строка)
  updated_at: string            // Дата обновления (ISO строка)
}
```

## Состояние компонента

### FinancePageContent

#### State переменные:
- `activeTab` ('grades' | 'currencies' | 'taxes'): текущая активная вкладка
  - Инициализация: определяется из URL параметра ?tab= или 'grades' по умолчанию
  - Обновление: при переключении вкладки или изменении URL параметра
  - Использование: определение, какой контент отображать

#### Hooks:
- `useSearchParams()`: получение параметров из URL
  - Используется для: чтения параметра ?tab=
- `useRouter()`: программная навигация
  - Используется для: обновления URL при переключении вкладок
- `useEffect()`: синхронизация activeTab с URL параметром
  - Зависимости: [tabParam]
  - Логика: обновляет activeTab при изменении tabParam в URL

### GradesSection

#### State переменные:
- `grades` (Grade[]): список грейдов
  - Инициализация: mockGrades
  - Обновление: при создании, редактировании или удалении грейда
- `isAdding` (boolean): флаг добавления нового грейда
  - Инициализация: false
  - Обновление: при открытии/закрытии формы добавления
- `selectedGradeName` (string): выбранное название грейда для добавления
  - Инициализация: ''
  - Обновление: при выборе грейда в Select
- `editingId` (number | null): ID редактируемого грейда
  - Инициализация: null
  - Обновление: при открытии/закрытии режима редактирования
- `editingName` (string): название грейда в режиме редактирования
  - Инициализация: ''
  - Обновление: при изменении названия в TextField

#### Hooks:
- `useToast()`: для отображения уведомлений при удалении

### CurrencyRatesSection

#### State переменные:
- `rates` (CurrencyRate[]): список курсов валют
  - Инициализация: mockRates
  - Обновление: при обновлении курсов валют
- `updating` (boolean): флаг обновления курсов валют
  - Инициализация: false
  - Обновление: при начале/завершении обновления

### TaxesSection

#### State переменные:
- `taxes` (PLNTax[]): список налогов
  - Инициализация: mockTaxes
  - Обновление: при создании, редактировании, удалении или переключении активности налога
- `isAdding` (boolean): флаг добавления нового налога
  - Инициализация: false
  - Обновление: при открытии/закрытии формы добавления
- `newTax` ({ name: string, rate: string }): данные нового налога для формы
  - Инициализация: { name: '', rate: '' }
  - Обновление: при изменении полей формы
- `editingId` (number | null): ID редактируемого налога
  - Инициализация: null
  - Обновление: при открытии/закрытии режима редактирования
- `editingTax` ({ name: string, rate: string, is_active: boolean }): данные налога в режиме редактирования
  - Инициализация: { name: '', rate: '', is_active: true }
  - Обновление: при изменении полей в режиме редактирования

#### Hooks:
- `useToast()`: для отображения уведомлений при удалении

## Функции и обработчики

### FinancePageContent

#### handleTabChange(value)
- **Назначение**: Обработчик переключения вкладки
- **Параметры**: 
  - `value`: string - значение новой вкладки ('grades', 'currencies', 'taxes')
- **Логика**: 
  1. Обновляет activeTab
  2. Обновляет URL параметр ?tab= через router.push()
  3. Отключает прокрутку страницы (scroll: false)
- **Поведение**: 
  - Вызывается при клике на вкладку
  - Обновляет URL для возможности делиться ссылками на конкретные вкладки
- **Связи**: 
  - Используется в Tabs.Root onValueChange

#### useEffect для синхронизации с URL
- **Назначение**: Синхронизация активной вкладки с URL параметром
- **Зависимости**: [tabParam]
- **Логика**: 
  1. Проверяет наличие tabParam в URL
  2. Валидирует значение (должно быть 'grades', 'currencies' или 'taxes')
  3. Обновляет activeTab если значение валидно
- **Поведение**: 
  - Выполняется при изменении tabParam
  - Позволяет открывать страницу с конкретной вкладкой через URL
  - Пример: /finance?tab=currencies откроет вкладку "Курсы валют"

### GradesSection

#### handleAdd()
- **Назначение**: Обработчик добавления нового грейда
- **Параметры**: Нет
- **Логика**: 
  1. Проверяет наличие selectedGradeName
  2. Создает новый грейд с уникальным ID
  3. Добавляет грейд в список
  4. Сбрасывает форму добавления
- **Поведение**: 
  - Вызывается при клике на кнопку "Сохранить" в форме добавления
  - Показывает alert при отсутствии выбранного грейда
- **Связи**: 
  - Используется в кнопке "Сохранить" формы добавления

#### handleEdit(grade)
- **Назначение**: Обработчик начала редактирования грейда
- **Параметры**: 
  - `grade`: Grade - грейд для редактирования
- **Логика**: 
  1. Устанавливает editingId в grade.id
  2. Устанавливает editingName в grade.name
- **Поведение**: 
  - Вызывается при клике на кнопку редактирования
  - Переводит карточку грейда в режим редактирования
- **Связи**: 
  - Используется в кнопке редактирования грейда

#### handleSaveEdit()
- **Назначение**: Обработчик сохранения изменений грейда
- **Параметры**: Нет
- **Логика**: 
  1. Проверяет наличие editingId и editingName
  2. Обновляет название грейда в списке
  3. Сбрасывает режим редактирования
- **Поведение**: 
  - Вызывается при клике на кнопку "Сохранить" в режиме редактирования
  - Показывает alert при отсутствии названия
  - Поддерживает Enter для сохранения, Escape для отмены
- **Связи**: 
  - Используется в кнопке "Сохранить" режима редактирования

#### handleDelete(id)
- **Назначение**: Обработчик удаления грейда
- **Параметры**: 
  - `id`: number - ID грейда для удаления
- **Логика**: 
  1. Показывает toast с предупреждением и кнопками "Отмена" и "Удалить"
  2. При подтверждении удаляет грейд из списка
- **Поведение**: 
  - Вызывается при клике на кнопку удаления
  - Показывает предупреждение перед удалением
  - Удаляет грейд после подтверждения
- **Связи**: 
  - Использует useToast для отображения предупреждения
  - Используется в кнопке удаления грейда

### CurrencyRatesSection

#### handleUpdateRates()
- **Назначение**: Обработчик обновления курсов валют
- **Параметры**: Нет
- **Логика**: 
  1. Устанавливает updating в true (показывает индикатор загрузки)
  2. Имитирует обновление (setTimeout)
  3. Обновляет дату получения курсов
  4. Обновляет статус курсов
  5. Устанавливает updating в false
  6. Показывает alert об успешном обновлении
- **Поведение**: 
  - Вызывается при клике на кнопку "Обновить валюты"
  - Показывает индикатор загрузки (анимация вращения иконки)
  - В текущей реализации только имитирует обновление
- **Связи**: 
  - Используется в кнопке "Обновить валюты"
- **TODO**: Реализовать реальное обновление через API

#### formatDate(dateString)
- **Назначение**: Форматирование даты для отображения
- **Параметры**: 
  - `dateString`: string - дата в формате ISO строки
- **Возвращает**: string - дата в формате "DD.MM.YYYY HH:MM"
- **Логика**: 
  - Использует toLocaleDateString с опциями для русского формата
- **Поведение**: 
  - Используется для отображения даты последнего обновления курсов
- **Связи**: 
  - Используется в таблице курсов валют

#### getStatusBadge(rate)
- **Назначение**: Получение Badge для статуса курса валюты
- **Параметры**: 
  - `rate`: CurrencyRate - курс валюты
- **Возвращает**: JSX элемент Badge
- **Логика**: 
  - BYN: синий Badge "Базовая валюта"
  - Обновлено менее 24 часов назад: зеленый Badge "Сегодня"
  - Обновлено менее 48 часов назад: оранжевый Badge "Вчера"
  - Обновлено более 48 часов назад: красный Badge "Устарело"
- **Поведение**: 
  - Используется в таблице курсов валют для отображения статуса
- **Связи**: 
  - Используется в таблице курсов валют

## Структура JSX

### FinancePage
- **Suspense**: обертка для FinancePageContent
  - `fallback`: AppLayout с текстом "Загрузка…"
  - Необходимо для корректной работы useSearchParams()

### FinancePageContent
- **AppLayout**: обертка страницы в общий layout
  - `pageTitle="Финансы и грейды"`
- **Box.container**: основной контейнер страницы
- **Flex direction="column" gap="4"**: контейнер для вкладок
  - **Tabs.Root**: компонент вкладок
    - `value={activeTab}`: текущая активная вкладка
    - `onValueChange={handleTabChange}`: обработчик переключения
    - **Tabs.List**: список вкладок
      - **Tabs.Trigger value="grades"**: вкладка "Грейды"
        - Иконка: StarIcon
      - **Tabs.Trigger value="currencies"**: вкладка "Курсы валют"
        - Иконка: ReloadIcon
      - **Tabs.Trigger value="taxes"**: вкладка "Налоги PLN"
        - Иконка: MixerHorizontalIcon
    - **Box.tabContent**: контейнер содержимого вкладок
      - **Tabs.Content value="grades"**: содержимое вкладки "Грейды"
        - GradesSection
      - **Tabs.Content value="currencies"**: содержимое вкладки "Курсы валют"
        - CurrencyRatesSection
      - **Tabs.Content value="taxes"**: содержимое вкладки "Налоги PLN"
        - TaxesSection

### GradesSection
- **Box.section**: основной контейнер секции
- **Flex.header**: заголовок секции
  - Иконка StarIcon и текст "Грейды сотрудников"
  - Кнопка "Добавить грейд"
- **Callout.Root.infoBox**: информационное сообщение
  - Текст о создании грейдов в настройках компании
- **Box.addForm**: форма добавления грейда (если isAdding === true)
  - Select для выбора грейда из доступных
  - Кнопки "Сохранить" и "Отмена"
- **Box.gradesGrid**: сетка карточек грейдов
  - Для каждого грейда:
    - **Box.gradeCard**: карточка грейда
      - Если editingId === grade.id: режим редактирования
        - TextField для редактирования названия
        - Кнопки "Сохранить" и "Отмена"
      - Иначе: режим просмотра
        - Иконка StarIcon и название грейда
        - Кнопки редактирования и удаления
        - Текст "Грейд сотрудника"

### CurrencyRatesSection
- **Box.section**: основной контейнер секции
- **Flex.header**: заголовок секции
  - Иконка ReloadIcon и текст "Курсы валют НБРБ"
  - Кнопка "Обновить валюты" (с анимацией вращения при обновлении)

## URL параметры

### ?tab=grades | ?tab=currencies | ?tab=taxes
- **Назначение**: Открытие конкретной вкладки при загрузке страницы
- **Логика**: 
  - Читается через useSearchParams().get('tab')
  - Валидируется перед установкой (должно быть 'grades', 'currencies' или 'taxes')
  - Используется для глубоких ссылок
- **Поведение**: 
  - При загрузке страницы с параметром ?tab= определяет активную вкладку
  - При переключении вкладки обновляет URL параметр
  - Позволяет делиться ссылками на конкретные вкладки
- **Пример**: 
  - `/finance?tab=currencies` откроет вкладку "Курсы валют"
  - `/finance?tab=taxes` откроет вкладку "Налоги PLN"

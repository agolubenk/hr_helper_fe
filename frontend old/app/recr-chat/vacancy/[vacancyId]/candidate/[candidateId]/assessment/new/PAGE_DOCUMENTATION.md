# Документация страницы: Создание оценки (recr-chat/.../assessment/new/page.tsx)

## Общее описание

Страница создания новой оценки кандидата по вакансии. Оценка вводится по блокам компетенций (аналогично матрице специализации). Варианты оценок в селектах берутся из шкалы, выбранной в матрице специализации, связанной с вакансией.

## Параметры маршрута

- `vacancyId` — идентификатор вакансии.
- `candidateId` — идентификатор кандидата.

## Компоненты страницы

- **Блоки компетенций** — группы (например: JS/TypeScript, React, Стили и сборка, Автономность и ответственность). В каждом блоке — список компетенций с Select по шкале оценок и кнопка «Скриншот».
- **Шкала оценок** — получается через `getRatingScaleForSpec(specId, 'technical')`. `specId` определяется по вакансии (мок: `VACANCY_SPEC_ID[vacancyId]`; далее — из API вакансии).
- **Ссылка на матрицу** — ведёт на `/specializations/${specId}/matrix` для контекста.

## Данные

- Блоки компетенций заданы моком `ASSESSMENT_BLOCKS` по аналогии с матрицей (технические + поведенческие).

## Связи с другими страницами

- `/company-settings/rating-scales` — справочник шкал.
- `/specializations/[id]/matrix` — выбор системы оценки в матрице определяет шкалу на этой странице.
- `/recr-chat/.../assessment/[assessmentId]/edit` — после создания доступно редактирование в течение ограниченного времени.

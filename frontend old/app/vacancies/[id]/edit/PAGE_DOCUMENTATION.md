# Документация страницы: Редактирование вакансии (vacancies/[id]/edit/page.tsx)

## Общее описание

Страница редактирования вакансии. Позволяет редактировать все параметры вакансии через форму с множеством разделов.

## Компоненты страницы

### 1. VacancyDetailHeader
- **Расположение**: `@/components/vacancies/VacancyDetailHeader`
- **Назначение**: Заголовок страницы с навигацией
- **Пропсы**:
  - `title`: название вакансии
  - `onBack`: обработчик возврата к списку
  - `onEdit`: обработчик редактирования (не используется, так как уже на странице редактирования)
- **Логика**: 
  - Отображает название вакансии
  - Кнопка "Назад к списку" возвращает на `/vacancies`
- **Связи**: Навигация на `/vacancies`

### 2. Разделы редактирования
- **Компоненты**: Различные секции редактирования
- **Назначение**: Редактирование разных аспектов вакансии
- **Разделы**:
  - Основная информация
  - Текст вакансии
  - Рекрутеры
  - Заказчики и интервьюеры
  - Вопросы и ссылки
  - Связи и интеграции
  - Статусы
  - Зарплатные вилки
  - Встречи и интервью
  - Scorecard
  - Обработка данных
  - История правок

## URL параметры

### [id]
- **Тип**: Динамический параметр маршрута
- **Назначение**: ID вакансии для редактирования
- **Пример**: `/vacancies/123/edit` редактирует вакансию с ID 123
- **Обработка**: 
  - Читается через `useParams()`
  - Используется для загрузки данных вакансии

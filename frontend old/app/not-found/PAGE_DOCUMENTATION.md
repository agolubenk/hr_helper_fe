# Документация страницы: Не найдено (not-found.tsx)

## Общее описание

Страница ошибки 404 (Not Found). Отображается при переходе на несуществующую страницу.

## Компоненты страницы

### 1. Сообщение об ошибке
- **Компонент**: `Text` с описанием
- **Назначение**: Информирование пользователя об ошибке
- **Логика**: 
  - Отображает код ошибки 404
  - Показывает сообщение "Страница не найдена"
  - Описание проблемы

### 2. Кнопка возврата
- **Компонент**: `Button` с навигацией
- **Назначение**: Возврат на главную страницу
- **Логика**: 
  - При клике происходит переход на `/`
  - Позволяет пользователю вернуться в приложение
- **Связи**: Навигация на `/`

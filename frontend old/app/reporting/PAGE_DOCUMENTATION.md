# Документация страницы: Отчетность (reporting/page.tsx)

## Общее описание

Главная страница отчетности. Отображает основные метрики и KPI (ключевые показатели эффективности) по рекрутингу в виде дашборда, предоставляет быстрый доступ к разделам отчетности и связанным страницам.

**Путь**: `/reporting`

## Компоненты страницы

### 1. ReportingPage (основной компонент)
- **Расположение**: `@/app/reporting/page.tsx`
- **Назначение**: Главный компонент страницы отчетности
- **Функциональность**: 
  - Отображение дашборда с основными метриками
  - Группировка метрик по категориям
  - Быстрые ссылки на разделы отчетности и связанные страницы
- **Связи**: 
  - AppLayout: оборачивает страницу в общий layout
  - useRouter: для навигации к разделам отчетности и связанным страницам
  - Sidebar: содержит ссылку на эту страницу

### 2. Заголовок страницы
- **Компонент**: Flex с иконкой и текстом
- **Назначение**: Заголовок раздела отчетности
- **Элементы**: 
  - Иконка BarChartIcon (24x24)
  - Текст "Отчетность" (size="6", weight="bold")

### 3. Секция HR метрик
- **Компонент**: Box с карточками метрик
- **Назначение**: Отображение основных HR метрик
- **Карточки**:
  - **Интервьюеры**: 
    - Иконка PersonIcon
    - Значение: "Активных / Всего" (например, "22 / 23")
    - Описание: "Активных из общего числа"
  - **Заявки**: 
    - Иконка FileTextIcon
    - Значение: общее количество заявок
    - Описание: разбивка по статусам (Планируется, В процессе, Закрыто)
  - **Вакансии**: 
    - Иконка FileTextIcon
    - Значение: "Активных / Всего" (например, "18 / 25")
    - Описание: "Активных из общего числа"

### 4. Секция SLA и скорость
- **Компонент**: Box с карточками метрик
- **Назначение**: Отображение метрик SLA и скорости процессов
- **Карточки**:
  - **Покрытие SLA**: 
    - Иконка ClockIcon
    - Значение: процент покрытия (например, "92%")
    - Прогресс-бар: визуальное отображение процента
    - Описание: "X из Y активных SLA"
  - **Средний Time-to-Offer**: 
    - Иконка ArrowUpIcon
    - Значение: среднее время до оффера в днях (например, "45 дней")
    - Описание: "Среднее время до оффера"
  - **Средний Time-to-Hire**: 
    - Иконка ArrowUpIcon
    - Значение: среднее время до найма в днях (например, "60 дней")
    - Описание: "Среднее время до найма"
  - **Средние дни рекрутера**: 
    - Иконка CalendarIcon
    - Значение: среднее время работы рекрутера в днях (например, "25 дней")
    - Описание: "Среднее время работы рекрутера"

### 5. Секция Исполнение
- **Компонент**: Box с карточками метрик
- **Назначение**: Отображение метрик исполнения заявок
- **Карточки**:
  - **В срок**: 
    - Иконка CheckCircledIcon (зеленая)
    - Значение: количество заявок в срок (например, "28")
    - Цвет значения: зеленый (#10b981)
    - Описание: "Заявок выполнено в срок"
  - **Просрочено**: 
    - Иконка ExclamationTriangleIcon (красная)
    - Значение: количество просроченных заявок (например, "10")
    - Цвет значения: красный (#ef4444)
    - Описание: "Заявок с просрочкой"
  - **Процент в срок**: 
    - Иконка ArrowUpIcon
    - Значение: процент заявок в срок (вычисляется: onTimeRequests / (onTimeRequests + overdueRequests) * 100)
    - Описание: "Доля заявок в срок"

### 6. Секция Результаты
- **Компонент**: Box с карточками метрик
- **Назначение**: Отображение метрик результатов рекрутинга
- **Карточки**:
  - **Всего кандидатов**: 
    - Иконка PersonIcon
    - Значение: общее количество кандидатов (например, "45")
    - Описание: "Кандидатов в системе"
  - **Нанято**: 
    - Иконка CheckCircledIcon (зеленая)
    - Значение: количество нанятых кандидатов (например, "12")
    - Цвет значения: зеленый (#10b981)
    - Описание: "Успешно нанятых кандидатов"
  - **Завершение интервью**: 
    - Иконка ArrowUpIcon
    - Значение: процент завершенных интервью (например, "85%")
    - Описание: "Процент завершенных интервью"
  - **Принятие офферов**: 
    - Иконка ArrowUpIcon
    - Значение: процент принятых офферов (например, "78%")
    - Описание: "Процент принятых офферов"

### 7. Быстрые ссылки
- **Компонент**: Box с кнопками навигации
- **Назначение**: Быстрый переход к разделам отчетности и связанным страницам
- **Кнопки**:
  - **Заявки**: переход на `/hiring-requests` (иконка FileTextIcon)
  - **Вакансии**: переход на `/vacancies` (иконка FileTextIcon)
  - **Интервьюеры**: переход на `/interviewers` (иконка PersonIcon)
  - **SLA**: переход на `/company-settings/sla` (иконка ClockIcon)
  - **Huntflow**: переход на `/huntflow` (иконка FileTextIcon)

## Интерфейсы и типы

### Metrics (метрики)
```typescript
interface Metrics {
  // HR метрики
  totalInterviewers: number              // Общее количество интервьюеров
  activeInterviewers: number            // Количество активных интервьюеров
  totalRequests: number                 // Общее количество заявок
  plannedRequests: number                // Количество запланированных заявок
  inProcessRequests: number              // Количество заявок в процессе
  closedRequests: number                // Количество закрытых заявок
  cancelledRequests: number             // Количество отмененных заявок
  totalVacancies: number                // Общее количество вакансий
  activeVacancies: number               // Количество активных вакансий
  inactiveVacancies: number             // Количество неактивных вакансий
  
  // SLA метрики
  totalSLAs: number                     // Общее количество SLA
  activeSLAs: number                    // Количество активных SLA
  slaCoverage: number                    // Процент покрытия SLA (0-100)
  
  // Скорость и исполнение
  avgTimeToOffer: number                // Среднее время до оффера (дни)
  avgTimeToHire: number                 // Среднее время до найма (дни)
  avgRecruiterDays: number              // Средние дни работы рекрутера
  onTimeRequests: number                 // Количество заявок в срок
  overdueRequests: number                // Количество просроченных заявок
  
  // Результаты
  totalCandidates: number               // Общее количество кандидатов
  hiredCandidates: number                // Количество нанятых кандидатов
  interviewCompletionRate: number         // Процент завершения интервью (0-100)
  offerAcceptanceRate: number            // Процент принятия офферов (0-100)
}
```

### Константы:

## Функции и обработчики

### Вычисляемые значения:
- **Процент в срок**: вычисляется как `Math.round((onTimeRequests / (onTimeRequests + overdueRequests)) * 100)`
  - Используется для: отображения процента заявок в срок в секции "Исполнение"
  - Логика: отношение заявок в срок к общему количеству заявок (в срок + просроченные)

### Обработчики:
- **onClick для быстрых ссылок**: 
  - `router.push('/hiring-requests')` - переход на страницу заявок
  - `router.push('/vacancies')` - переход на страницу вакансий
  - `router.push('/interviewers')` - переход на страницу интервьюеров
  - `router.push('/company-settings/sla')` - переход на страницу SLA
  - `router.push('/huntflow')` - переход на страницу Huntflow

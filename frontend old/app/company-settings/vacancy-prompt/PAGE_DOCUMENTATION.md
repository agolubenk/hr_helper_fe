# Документация страницы: Промпты вакансий (company-settings/vacancy-prompt/page.tsx)

## Общее описание

Страница настройки промптов для вакансий. Позволяет настраивать шаблоны промптов для генерации описаний вакансий и других текстов.

## Компоненты страницы

### 1. VacancyPromptSettings
- **Расположение**: `@/components/company-settings/VacancyPromptSettings`
- **Назначение**: Компонент настроек промптов вакансий
- **Пропсы**: Нет (использует внутреннее состояние)
- **Логика**: 
  - Отображение списка шаблонов промптов
  - Создание новых шаблонов
  - Редактирование существующих шаблонов
  - Удаление шаблонов
  - Настройка параметров промптов
- **Связи**: Используется для управления промптами вакансий

## Состояние компонента

### State переменные:
- `prompts`: массив всех шаблонов промптов
- `editingPrompt`: промпт для редактирования
- `showAddForm`: флаг отображения формы добавления

## Функции и обработчики

### CRUD операции:
- **Создание**: Добавление нового шаблона промпта
- **Чтение**: Загрузка списка шаблонов
- **Обновление**: Редактирование существующего шаблона
- **Удаление**: Удаление шаблона с подтверждением

## Моковые данные

### mockPrompts
Массив объектов шаблонов промптов с полями:
- `id`: уникальный идентификатор
- `name`: название шаблона
- `template`: шаблон промпта
- `variables`: переменные для подстановки
- `description`: описание шаблона

# Документация страницы: Жизненный цикл сотрудника (company-settings/employee-lifecycle/page.tsx)

## Общее описание

Страница настройки жизненного цикла сотрудника. Позволяет настраивать этапы жизненного цикла сотрудника от найма до увольнения.

## Компоненты страницы

### 1. EmployeeLifecycleSettings
- **Расположение**: `@/components/company-settings/EmployeeLifecycleSettings`
- **Назначение**: Компонент настроек жизненного цикла сотрудника
- **Пропсы**: Нет (использует внутреннее состояние)
- **Логика**: 
  - Отображение этапов жизненного цикла
  - Создание новых этапов
  - Редактирование существующих этапов
  - Удаление этапов
  - Настройка параметров этапов
- **Связи**: Используется для управления жизненным циклом сотрудника

## Состояние компонента

### State переменные:
- `lifecycleStages`: массив всех этапов жизненного цикла
- `editingStage`: этап для редактирования
- `showAddForm`: флаг отображения формы добавления

## Функции и обработчики

### CRUD операции:
- **Создание**: Добавление нового этапа
- **Чтение**: Загрузка списка этапов
- **Обновление**: Редактирование существующего этапа
- **Удаление**: Удаление этапа с подтверждением
- **Переупорядочивание**: Изменение порядка этапов

## Моковые данные

### mockLifecycleStages
Массив объектов этапов жизненного цикла с полями:
- `id`: уникальный идентификатор
- `name`: название этапа (Найм, Онбординг, Работа, Увольнение)
- `order`: порядок следования
- `description`: описание этапа
- `parameters`: параметры этапа

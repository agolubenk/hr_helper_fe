# Документация страницы: Финансовые настройки (company-settings/finance/page.tsx)

## Общее описание

Страница финансовых настроек компании. Дублирует функционал страницы `/finance`, но находится в разделе настроек компании.

## Компоненты страницы

### 1. GradesSection
- **Расположение**: `@/components/finance/GradesSection`
- **Назначение**: Настройка грейдов и уровней
- **Пропсы**: Нет (использует внутреннее состояние)
- **Логика**: Аналогична странице `/finance`
- **Связи**: Используется для настройки грейдов

### 2. CurrencyRatesSection
- **Расположение**: `@/components/finance/CurrencyRatesSection`
- **Назначение**: Управление курсами валют
- **Пропсы**: Нет (использует внутреннее состояние)
- **Логика**: Аналогична странице `/finance`
- **Связи**: Используется для настройки курсов валют

### 3. TaxesSection
- **Расположение**: `@/components/finance/TaxesSection`
- **Назначение**: Настройка налогов
- **Пропсы**: Нет (использует внутреннее состояние)
- **Логика**: Аналогична странице `/finance`
- **Связи**: Используется для настройки налогов

## Состояние компонента

### State переменные:
- `activeTab`: текущая активная вкладка ('grades' | 'currencies' | 'taxes')

## Функции и обработчики

### Аналогичны странице `/finance`
- Переключение вкладок
- Загрузка и сохранение данных
- Управление грейдами, курсами валют и налогами

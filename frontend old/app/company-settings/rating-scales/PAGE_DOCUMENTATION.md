# Документация страницы: Шкалы оценок (company-settings/rating-scales/page.tsx)

## Общее описание

Страница управления шкалами оценок компании. Шкалы используются при проведении интервью и в матрице компетенций специализации. Шкала по умолчанию подставляется при создании новой оценки кандидата.

## Компоненты страницы

### 1. RatingScalesSettings
- **Расположение**: `@/components/company-settings/RatingScalesSettings`
- **Назначение**: Список шкал, создание/редактирование/удаление, выбор шкалы по умолчанию
- **Логика**:
  - Отображение списка шкал (название, пункты шкалы)
  - Модальное окно для добавления/редактирования шкалы (название + пункты: значение и подпись)
  - Отметка шкалы по умолчанию
- **Связи**: Данные шкал используются в матрице специализации (выбор «Система оценки» в табах) и на странице создания/редактирования оценки

## Состояние

Управление состоянием списка шкал и модалок выполняется внутри компонента `RatingScalesSettings`.

## Моковые данные

- Шкалы задаются в `lib/ratingScale.ts` (getRatingScales): например «1–5 баллов», «Сдал / Не сдал», «Четыре уровня».

## Связи с другими страницами

- `/specializations/[id]/matrix` — в начале каждого таба (технические/поведенческие/задачи/тесты) выбирается система оценки из этих шкал; выбор сохраняется по специализации и табу.
- `/recr-chat/.../assessment/new` и `.../assessment/[id]/edit` — варианты оценок в селектах берутся из шкалы, выбранной в матрице специализации, связанной с вакансией.

## Особенности

- Шкалы задают допустимые значения при оценке компетенций (баллы или категории).
- Одна из шкал может быть помечена как используемая по умолчанию при создании новой оценки.

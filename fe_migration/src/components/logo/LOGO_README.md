# Логотип приложения (робот) — единый источник

## Обязательное правило (не нарушать)

**Логотип-робот определён только здесь.** Все места в приложении, где отображается логотип (хедер, страница входа, фавикон, страницы ошибок, виджеты и т.д.), **обязаны** использовать компонент и утилиты из этой папки:

- **Компонент:** `LogoRobot` — импорт: `import { LogoRobot } from '@/components/logo'`
- **Цвета:** `logoColors.ts` — тип `AccentColorValue`, функция `getLogoColors(accentColor, theme)`
- **Фавикон:** `getFaviconDataUrl` из `@/components/logo` (использует те же цвета и SVG)

**Запрещено:**

- Дублировать SVG робота в других файлах или проектах.
- Создавать альтернативные компоненты логотипа без согласования.
- Хардкодить пути к картинкам логотипа вне этого модуля.

Так мы улучшаем лого в одном месте, и качественно он отображается везде. При изменении дизайна логотипа правки вносятся только в `LogoRobot.tsx` и при необходимости в `logoColors.ts`.

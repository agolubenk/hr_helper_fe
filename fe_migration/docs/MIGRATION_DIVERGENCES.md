# Расхождения fe_migration ↔ эталон (Next / `frontend old`)

**Правило команды:** намеренные изменения из этой ветки **не откатывать**. Если при сравнении с **3001** (или с `frontend old`) что-то **не совпадает** — **не «чинить под эталон» автоматически**: занести сюда строку (раздел, кратко «было / стало / файл»). Исправление под эталон — только по явному запросу в задаче.

**Связанные документы:** обзор UI-профиля и вики — `docs/UI_MIGRATION_PLAN.md`; детали вики — `src/components/wiki/COMPONENTS_DOCUMENTATION.md`; чек-URL — `docs/MIGRATION_CHECK_URLS.md`.

## 6. Вики — редактирование (`/wiki/:id/edit`, компоненты формы)

| Тема | Отличие | Где |
|------|---------|-----|
| Шаблон страницы | Убран **верхний селект** «Шаблон страницы»; шаблоны только через **«Добавить шаблон»** и модалку | `WikiEditForm`, `COMPONENTS_DOCUMENTATION.md` |
| Карточки | Три карточки на всю ширину: Содержание, Параметры, Действия (компоновка ≠ старый Next, если там было иначе) | `WikiEditForm` |
| Справка на мобильных | Справка в сайдбаре скрыта при **≤768px**; кнопка **Info** в шапке → модалка с тем же текстом | `WikiEditForm` |
| Плавающие действия | Компонент **`FloatingConfirmActions`**: позиция по `[data-app-layout-content]`, IntersectionObserver / `--expansion`, скрытие дублей кнопок в карточке | `FloatingConfirmActions.tsx` |
| Мобильный режим формы | Порог **ширины блока** карточки действий **800px** (ResizeObserver), не viewport | см. документацию вики |
| Удаление страницы | Подтверждение через **тост** (Отмена / Удалить), не Popover | `WikiDeleteConfirmDialog` / toast |

## 7. Профиль, футер, тема, плавающая панель

Сводка из **`docs/UI_MIGRATION_PLAN.md`** (не дублировать полностью — при расхождении с 3001 дополнять §9):

- Профиль: ФИО одной строкой, мок телефона, блок «Активная среда» убран, email вне блока соцсетей.
- UserCard: убрана строка Telegram в карточке; аватар/заглушка с рамкой и тенью.
- Соцсети: лимит **3 аккаунта на платформу**, тултипы при ≥2 иконках, копирование в углу иконки.
- Тема: Авто / Свет / Тёмная + акценты, `ThemeProvider`.
- Футер: фиксированный, трей задач, связка с `QuickTasksSettings`.
- **FloatingActions**: выравнивание с основным frontend (позиции, стекло, порядок кнопок, `quickButtonsStorage`).

## 10. Специализации / админка / ATS (кратко)
